# radar_radar_options

Options to configure Radar. See [Radar Session](https://docs.stripe.com/radar/radar-session) for more information.

**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `session` | string | No | A [Radar Session](https://docs.stripe.com/radar/radar-session) is a snapshot of the browser metadata and device details that help Radar make more accurate predictions on your payments. |
