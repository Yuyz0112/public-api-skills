# insights_resources_payment_evaluation_customer_details

Customer details attached to this payment evaluation.

**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `customer` | string | No | The ID of the customer associated with the payment evaluation. |
| `customer_account` | string | No | The ID of the Account representing the customer associated with the payment evaluation. |
| `email` | string | No | The customer's email address. |
| `name` | string | No | The customer's full name or business name. |
| `phone` | string | No | The customer's phone number. |
