# confirmation_tokens_resource_mandate_data_resource_customer_acceptance

This hash contains details about the customer acceptance of the Mandate.

**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `online` | any | No | If this is a Mandate accepted online, this hash contains details about the online acceptance. |
| `type` | string | Yes | The type of customer acceptance information included with the Mandate. |
