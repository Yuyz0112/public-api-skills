# account_dashboard_settings

**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `display_name` | string | No | The display name for this account. This is used on the Stripe Dashboard to differentiate between accounts. |
| `timezone` | string | No | The timezone used in the Stripe Dashboard for this account. A list of possible time zone values is maintained at the [IANA Time Zone Database](http://www.iana.org/time-zones). |
