# POST /admin.teams.settings.setIcon

**Resource:** [admin.teams.settings](../resources/admin-teams-settings.md)
**Operation ID:** `admin_teams_settings_setIcon`

**Required:** Yes

**Content Types:** `application/x-www-form-urlencoded`

## Responses
