# terminal_reader_reader_resource_choice

Choice to be selected on a Reader

**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `id` | string | No | The identifier for the selected choice. Maximum 50 characters. |
| `style` | enum: primary, secondary | No | The button style for the choice. Can be `primary` or `secondary`. |
| `text` | string | Yes | The text to be selected. Maximum 30 characters. |
