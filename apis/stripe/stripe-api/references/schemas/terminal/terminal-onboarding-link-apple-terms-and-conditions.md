# terminal_onboarding_link_apple_terms_and_conditions

Options associated with the Apple Terms and Conditions link type.

**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `allow_relinking` | boolean | No | Whether the link should also support users relinking their Apple account. |
| `merchant_display_name` | string | Yes | The business name of the merchant accepting Apple's Terms and Conditions. |
