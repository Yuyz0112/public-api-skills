# terminal_reader_reader_resource_custom_text

Represents custom text to be displayed when collecting the input using a reader

**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `description` | string | No | Customize the default description for this input |
| `skip_button` | string | No | Customize the default label for this input's skip button |
| `submit_button` | string | No | Customize the default label for this input's submit button |
| `title` | string | No | Customize the default title for this input |
