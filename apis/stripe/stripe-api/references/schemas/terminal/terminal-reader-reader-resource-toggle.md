# terminal_reader_reader_resource_toggle

Information about an input's toggle

**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `default_value` | enum: disabled, enabled | No | The toggle's default value. Can be `enabled` or `disabled`. |
| `description` | string | No | The toggle's description text. Maximum 50 characters. |
| `title` | string | No | The toggle's title text. Maximum 50 characters. |
| `value` | enum: disabled, enabled | No | The toggle's collected value. Can be `enabled` or `disabled`. |
