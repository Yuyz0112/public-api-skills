# Project roles

This resource represents the roles that users can play in projects. Use this resource to get, create, update, and delete project roles.

## Operations

| Method | Path | Summary | Details |
|--------|------|---------|----------|
| GET | `/rest/api/3/project/{projectIdOrKey}/role` | Get project roles for project | [View](../operations/getProjectRoles.md) |
| GET | `/rest/api/3/project/{projectIdOrKey}/role/{id}` | Get project role for project | [View](../operations/getProjectRole.md) |
| GET | `/rest/api/3/project/{projectIdOrKey}/roledetails` | Get project role details | [View](../operations/getProjectRoleDetails.md) |
| GET | `/rest/api/3/role` | Get all project roles | [View](../operations/getAllProjectRoles.md) |
| POST | `/rest/api/3/role` | Create project role | [View](../operations/createProjectRole.md) |
| GET | `/rest/api/3/role/{id}` | Get project role by ID | [View](../operations/getProjectRoleById.md) |
| PUT | `/rest/api/3/role/{id}` | Fully update project role | [View](../operations/fullyUpdateProjectRole.md) |
| POST | `/rest/api/3/role/{id}` | Partial update project role | [View](../operations/partialUpdateProjectRole.md) |
| DELETE | `/rest/api/3/role/{id}` | Delete project role | [View](../operations/deleteProjectRole.md) |
