# Project avatars

This resource represents avatars associated with a project. Use it to get, load, set, and remove project avatars.

## Operations

| Method | Path | Summary | Details |
|--------|------|---------|----------|
| PUT | `/rest/api/3/project/{projectIdOrKey}/avatar` | Set project avatar | [View](../operations/updateProjectAvatar.md) |
| DELETE | `/rest/api/3/project/{projectIdOrKey}/avatar/{id}` | Delete project avatar | [View](../operations/deleteProjectAvatar.md) |
| POST | `/rest/api/3/project/{projectIdOrKey}/avatar2` | Load project avatar | [View](../operations/createProjectAvatar.md) |
| GET | `/rest/api/3/project/{projectIdOrKey}/avatars` | Get all project avatars | [View](../operations/getAllProjectAvatars.md) |
