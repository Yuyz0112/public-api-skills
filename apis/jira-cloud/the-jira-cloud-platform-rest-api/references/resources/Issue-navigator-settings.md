# Issue navigator settings

This resource represents issue navigator settings. Use it to get and set issue navigator default columns.

| Method | Path | Summary | Details |
|--------|------|---------|----------|
| GET | `/rest/api/3/settings/columns` | Get issue navigator default columns | [View](../operations/getIssueNavigatorDefaultColumns.md) |
| PUT | `/rest/api/3/settings/columns` | Set issue navigator default columns | [View](../operations/setIssueNavigatorDefaultColumns.md) |
