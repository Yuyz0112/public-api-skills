# Issue votes

This resource represents votes cast by users on an issue. Use it to get details of votes on an issue as well as cast and withdrawal votes.

## Operations

| Method | Path | Summary | Details |
|--------|------|---------|----------|
| GET | `/rest/api/3/issue/{issueIdOrKey}/votes` | Get votes | [View](../operations/getVotes.md) |
| POST | `/rest/api/3/issue/{issueIdOrKey}/votes` | Add vote | [View](../operations/addVote.md) |
| DELETE | `/rest/api/3/issue/{issueIdOrKey}/votes` | Delete vote | [View](../operations/removeVote.md) |
