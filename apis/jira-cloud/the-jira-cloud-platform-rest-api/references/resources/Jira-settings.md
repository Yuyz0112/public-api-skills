# Jira settings

This resource represents various settings in Jira. Use it to get and update Jira settings and properties.

## Operations

| Method | Path | Summary | Details |
|--------|------|---------|----------|
| GET | `/rest/api/3/application-properties` | Get application property | [View](../operations/getApplicationProperty.md) |
| GET | `/rest/api/3/application-properties/advanced-settings` | Get advanced settings | [View](../operations/getAdvancedSettings.md) |
| PUT | `/rest/api/3/application-properties/{id}` | Set application property | [View](../operations/setApplicationProperty.md) |
| GET | `/rest/api/3/configuration` | Get global settings | [View](../operations/getConfiguration.md) |
