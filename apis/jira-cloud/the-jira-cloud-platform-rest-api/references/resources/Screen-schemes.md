# Screen schemes

This resource represents screen schemes in classic projects. Use it to get, create, update, and delete screen schemes.

## Operations

| Method | Path | Summary | Details |
|--------|------|---------|----------|
| GET | `/rest/api/3/screenscheme` | Get screen schemes | [View](../operations/getScreenSchemes.md) |
| POST | `/rest/api/3/screenscheme` | Create screen scheme | [View](../operations/createScreenScheme.md) |
| PUT | `/rest/api/3/screenscheme/{screenSchemeId}` | Update screen scheme | [View](../operations/updateScreenScheme.md) |
| DELETE | `/rest/api/3/screenscheme/{screenSchemeId}` | Delete screen scheme | [View](../operations/deleteScreenScheme.md) |
