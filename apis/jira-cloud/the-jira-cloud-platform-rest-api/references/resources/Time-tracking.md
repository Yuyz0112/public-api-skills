# Time tracking

This resource represents time tracking and time tracking providers. Use it to get and set the time tracking provider, get and set the time tracking options, and disable time tracking.

## Operations

| Method | Path | Summary | Details |
|--------|------|---------|----------|
| GET | `/rest/api/3/configuration/timetracking` | Get selected time tracking provider | [View](../operations/getSelectedTimeTrackingImplementation.md) |
| PUT | `/rest/api/3/configuration/timetracking` | Select time tracking provider | [View](../operations/selectTimeTrackingImplementation.md) |
| GET | `/rest/api/3/configuration/timetracking/list` | Get all time tracking providers | [View](../operations/getAvailableTimeTrackingImplementations.md) |
| GET | `/rest/api/3/configuration/timetracking/options` | Get time tracking settings | [View](../operations/getSharedTimeTrackingConfiguration.md) |
| PUT | `/rest/api/3/configuration/timetracking/options` | Set time tracking settings | [View](../operations/setSharedTimeTrackingConfiguration.md) |
