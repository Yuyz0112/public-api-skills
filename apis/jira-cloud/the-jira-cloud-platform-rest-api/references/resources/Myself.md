# Myself

This resource represents information about the current user, such as basic details, group membership, application roles, preferences, and locale. Use it to get, create, update, and delete (restore default) values of the user's preferences and locale.

## Operations

| Method | Path | Summary | Details |
|--------|------|---------|----------|
| GET | `/rest/api/3/mypreferences` | Get preference | [View](../operations/getPreference.md) |
| PUT | `/rest/api/3/mypreferences` | Set preference | [View](../operations/setPreference.md) |
| DELETE | `/rest/api/3/mypreferences` | Delete preference | [View](../operations/removePreference.md) |
| GET | `/rest/api/3/mypreferences/locale` | Get locale | [View](../operations/getLocale.md) |
| PUT | `/rest/api/3/mypreferences/locale` | Set locale | [View](../operations/setLocale.md) |
| GET | `/rest/api/3/myself` | Get current user | [View](../operations/getCurrentUser.md) |
