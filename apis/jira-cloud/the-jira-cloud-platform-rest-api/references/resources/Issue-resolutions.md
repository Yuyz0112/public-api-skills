# Issue resolutions

This resource represents issue resolution values. Use it to obtain a list of all issue resolution values and the details of individual resolution values.

## Operations

| Method | Path | Summary | Details |
|--------|------|---------|----------|
| GET | `/rest/api/3/resolution` | Get resolutions | [View](../operations/getResolutions.md) |
| POST | `/rest/api/3/resolution` | Create resolution | [View](../operations/createResolution.md) |
| PUT | `/rest/api/3/resolution/default` | Set default resolution | [View](../operations/setDefaultResolution.md) |
| PUT | `/rest/api/3/resolution/move` | Move resolutions | [View](../operations/moveResolutions.md) |
| GET | `/rest/api/3/resolution/search` | Search resolutions | [View](../operations/searchResolutions.md) |
| GET | `/rest/api/3/resolution/{id}` | Get resolution | [View](../operations/getResolution.md) |
| PUT | `/rest/api/3/resolution/{id}` | Update resolution | [View](../operations/updateResolution.md) |
| DELETE | `/rest/api/3/resolution/{id}` | Delete resolution | [View](../operations/deleteResolution.md) |
