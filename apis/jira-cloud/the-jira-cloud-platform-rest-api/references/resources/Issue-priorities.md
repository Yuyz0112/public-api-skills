# Issue priorities

This resource represents issue priorities. Use it to get, create and update issue priorities and details for individual issue priorities.

## Operations

| Method | Path | Summary | Details |
|--------|------|---------|----------|
| GET | `/rest/api/3/priority` | Get priorities | [View](../operations/getPriorities.md) |
| POST | `/rest/api/3/priority` | Create priority | [View](../operations/createPriority.md) |
| PUT | `/rest/api/3/priority/default` | Set default priority | [View](../operations/setDefaultPriority.md) |
| PUT | `/rest/api/3/priority/move` | Move priorities | [View](../operations/movePriorities.md) |
| GET | `/rest/api/3/priority/search` | Search priorities | [View](../operations/searchPriorities.md) |
| GET | `/rest/api/3/priority/{id}` | Get priority | [View](../operations/getPriority.md) |
| PUT | `/rest/api/3/priority/{id}` | Update priority | [View](../operations/updatePriority.md) |
| DELETE | `/rest/api/3/priority/{id}` | Delete priority | [View](../operations/deletePriority.md) |
