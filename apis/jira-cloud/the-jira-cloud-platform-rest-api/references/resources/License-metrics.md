# License metrics

This resource represents license metrics. Use it to get available metrics for Jira licences.

## Operations

| Method | Path | Summary | Details |
|--------|------|---------|----------|
| GET | `/rest/api/3/instance/license` | Get license | [View](../operations/getLicense.md) |
| GET | `/rest/api/3/license/approximateLicenseCount` | Get approximate license count | [View](../operations/getApproximateLicenseCount.md) |
| GET | `/rest/api/3/license/approximateLicenseCount/product/{applicationKey}` | Get approximate application license count | [View](../operations/getApproximateApplicationLicenseCount.md) |
