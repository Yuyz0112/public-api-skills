# Issue custom field associations

This resource represents the fields associated to project and issue type contexts. Use it to:

 *  assign custom field to projects and issue types.

## Operations

| Method | Path | Summary | Details |
|--------|------|---------|----------|
| PUT | `/rest/api/3/field/association` | Create associations | [View](../operations/createAssociations.md) |
| DELETE | `/rest/api/3/field/association` | Remove associations | [View](../operations/removeAssociations.md) |
