# Permissions

This resource represents permissions. Use it to obtain details of all permissions and determine whether the user has certain permissions.

## Operations

| Method | Path | Summary | Details |
|--------|------|---------|----------|
| GET | `/rest/api/3/mypermissions` | Get my permissions | [View](../operations/getMyPermissions.md) |
| GET | `/rest/api/3/permissions` | Get all permissions | [View](../operations/getAllPermissions.md) |
| POST | `/rest/api/3/permissions/check` | Get bulk permissions | [View](../operations/getBulkPermissions.md) |
| POST | `/rest/api/3/permissions/project` | Get permitted projects | [View](../operations/getPermittedProjects.md) |
