# Project types

This resource represents project types. Use it to obtain a list of all project types, a list of project types accessible to the calling user, and details of a project type.

## Operations

| Method | Path | Summary | Details |
|--------|------|---------|----------|
| GET | `/rest/api/3/project/type` | Get all project types | [View](../operations/getAllProjectTypes.md) |
| GET | `/rest/api/3/project/type/accessible` | Get licensed project types | [View](../operations/getAllAccessibleProjectTypes.md) |
| GET | `/rest/api/3/project/type/{projectTypeKey}` | Get project type by key | [View](../operations/getProjectTypeByKey.md) |
| GET | `/rest/api/3/project/type/{projectTypeKey}/accessible` | Get accessible project type by key | [View](../operations/getAccessibleProjectTypeByKey.md) |
