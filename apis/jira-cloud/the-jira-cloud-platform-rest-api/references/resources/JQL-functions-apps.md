# JQL functions (apps)

This resource represents JQL function's precomputations. Precomputation is a mapping between custom function call and JQL fragment returned by this function. Use it to get and update precomputations.

## Operations

| Method | Path | Summary | Details |
|--------|------|---------|----------|
| GET | `/rest/api/3/jql/function/computation` | Get precomputations (apps) | [View](../operations/getPrecomputations.md) |
| POST | `/rest/api/3/jql/function/computation` | Update precomputations (apps) | [View](../operations/updatePrecomputations.md) |
| POST | `/rest/api/3/jql/function/computation/search` | Get precomputations by ID (apps) | [View](../operations/getPrecomputationsByID.md) |
