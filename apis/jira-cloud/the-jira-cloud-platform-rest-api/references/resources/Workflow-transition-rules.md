# Workflow transition rules

This resource represents workflow transition rules. Workflow transition rules define a Connect or a Forge app routine, such as a [workflow post functions](https://developer.atlassian.com/cloud/jira/platform/modules/workflow-post-function/) that is executed in association with the workflow. Use it to read and modify configuration of workflow transition rules.

## Operations

| Method | Path | Summary | Details |
|--------|------|---------|----------|
| GET | `/rest/api/3/workflow/rule/config` | Get workflow transition rule configurations | [View](../operations/getWorkflowTransitionRuleConfigurations.md) |
| PUT | `/rest/api/3/workflow/rule/config` | Update workflow transition rule configurations | [View](../operations/updateWorkflowTransitionRuleConfigurations.md) |
| PUT | `/rest/api/3/workflow/rule/config/delete` | Delete workflow transition rule configurations | [View](../operations/deleteWorkflowTransitionRuleConfigurations.md) |
