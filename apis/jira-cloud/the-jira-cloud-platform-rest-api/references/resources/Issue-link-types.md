# Issue link types

This resource represents [issue link](#api-group-Issue-links) types. Use it to get, create, update, and delete link issue types as well as get lists of all link issue types.

To use it, the site must have [issue linking](https://confluence.atlassian.com/x/yoXKM) enabled.

## Operations

| Method | Path | Summary | Details |
|--------|------|---------|----------|
| GET | `/rest/api/3/issueLinkType` | Get issue link types | [View](../operations/getIssueLinkTypes.md) |
| POST | `/rest/api/3/issueLinkType` | Create issue link type | [View](../operations/createIssueLinkType.md) |
| GET | `/rest/api/3/issueLinkType/{issueLinkTypeId}` | Get issue link type | [View](../operations/getIssueLinkType.md) |
| PUT | `/rest/api/3/issueLinkType/{issueLinkTypeId}` | Update issue link type | [View](../operations/updateIssueLinkType.md) |
| DELETE | `/rest/api/3/issueLinkType/{issueLinkTypeId}` | Delete issue link type | [View](../operations/deleteIssueLinkType.md) |
