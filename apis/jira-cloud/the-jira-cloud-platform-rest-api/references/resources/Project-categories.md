# Project categories

This resource represents project categories. Use it to create, update, and delete project categories as well as obtain a list of all project categories and details of individual categories. For more information on managing project categories, see [Adding, assigning, and deleting project categories](https://confluence.atlassian.com/x/-A5WMg).

## Operations

| Method | Path | Summary | Details |
|--------|------|---------|----------|
| GET | `/rest/api/3/projectCategory` | Get all project categories | [View](../operations/getAllProjectCategories.md) |
| POST | `/rest/api/3/projectCategory` | Create project category | [View](../operations/createProjectCategory.md) |
| GET | `/rest/api/3/projectCategory/{id}` | Get project category by ID | [View](../operations/getProjectCategoryById.md) |
| PUT | `/rest/api/3/projectCategory/{id}` | Update project category | [View](../operations/updateProjectCategory.md) |
| DELETE | `/rest/api/3/projectCategory/{id}` | Delete project category | [View](../operations/removeProjectCategory.md) |
