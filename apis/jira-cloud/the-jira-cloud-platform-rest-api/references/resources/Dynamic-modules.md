# Dynamic modules

This resource represents [modules registered dynamically](https://developer.atlassian.com/cloud/jira/platform/dynamic-modules/)
by [Connect apps](https://developer.atlassian.com/cloud/jira/platform/index/#connect-apps).

## Operations

| Method | Path | Summary | Details |
|--------|------|---------|----------|
| GET | `/rest/atlassian-connect/1/app/module/dynamic` | Get modules | [View](../operations/DynamicModulesResource-getModules-get.md) |
| POST | `/rest/atlassian-connect/1/app/module/dynamic` | Register modules | [View](../operations/DynamicModulesResource-registerModules-post.md) |
| DELETE | `/rest/atlassian-connect/1/app/module/dynamic` | Remove modules | [View](../operations/DynamicModulesResource-removeModules-delete.md) |
