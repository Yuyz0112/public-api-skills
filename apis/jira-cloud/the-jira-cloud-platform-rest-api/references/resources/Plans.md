# Plans

This resource represents plans. Use it to get, create, duplicate, update, trash and archive plans.

## Operations

| Method | Path | Summary | Details |
|--------|------|---------|----------|
| GET | `/rest/api/3/plans/plan` | Get plans paginated | [View](../operations/getPlans.md) |
| POST | `/rest/api/3/plans/plan` | Create plan | [View](../operations/createPlan.md) |
| GET | `/rest/api/3/plans/plan/{planId}` | Get plan | [View](../operations/getPlan.md) |
| PUT | `/rest/api/3/plans/plan/{planId}` | Update plan | [View](../operations/updatePlan.md) |
| PUT | `/rest/api/3/plans/plan/{planId}/archive` | Archive plan | [View](../operations/archivePlan.md) |
| POST | `/rest/api/3/plans/plan/{planId}/duplicate` | Duplicate plan | [View](../operations/duplicatePlan.md) |
| PUT | `/rest/api/3/plans/plan/{planId}/trash` | Trash plan | [View](../operations/trashPlan.md) |
