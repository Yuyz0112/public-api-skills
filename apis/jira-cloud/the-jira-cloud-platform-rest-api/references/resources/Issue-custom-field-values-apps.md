# Issue custom field values (apps)

This resource represents the values of custom fields added by [Forge apps](https://developer.atlassian.com/platform/forge/). Use it to update the value of a custom field on issues.

## Operations

| Method | Path | Summary | Details |
|--------|------|---------|----------|
| POST | `/rest/api/3/app/field/value` | Update custom fields | [View](../operations/updateMultipleCustomFieldValues.md) |
| PUT | `/rest/api/3/app/field/{fieldIdOrKey}/value` | Update custom field value | [View](../operations/updateCustomFieldValue.md) |
