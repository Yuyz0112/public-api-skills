# Project templates

This resource represents project templates. Use it to create a new project from a custom template.

## Operations

| Method | Path | Summary | Details |
|--------|------|---------|----------|
| POST | `/rest/api/3/project-template` | Create custom project | [View](../operations/createProjectWithCustomTemplate.md) |
| PUT | `/rest/api/3/project-template/edit-template` | Edit a custom project template | [View](../operations/editTemplate.md) |
| GET | `/rest/api/3/project-template/live-template` | Gets a custom project template | [View](../operations/liveTemplate.md) |
| DELETE | `/rest/api/3/project-template/remove-template` | Deletes a custom project template | [View](../operations/removeTemplate.md) |
| POST | `/rest/api/3/project-template/save-template` | Save a custom project template | [View](../operations/saveTemplate.md) |
