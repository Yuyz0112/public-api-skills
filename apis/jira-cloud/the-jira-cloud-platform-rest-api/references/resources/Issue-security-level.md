# Issue security level

This resource represents issue security levels. Use it to obtain the details of any issue security level. For more information about issue security levels, see [Configuring issue-level security](https://confluence.atlassian.com/x/J4lKLg).

## Operations

| Method | Path | Summary | Details |
|--------|------|---------|----------|
| GET | `/rest/api/3/issuesecurityschemes/{issueSecuritySchemeId}/members` | Get issue security level members by issue security scheme | [View](../operations/getIssueSecurityLevelMembers.md) |
| GET | `/rest/api/3/securitylevel/{id}` | Get issue security level | [View](../operations/getIssueSecurityLevel.md) |
