# Issue watchers

This resource represents users watching an issue. Use it to get details of users watching an issue as well as start and stop a user watching an issue.

## Operations

| Method | Path | Summary | Details |
|--------|------|---------|----------|
| POST | `/rest/api/3/issue/watching` | Get is watching issue bulk | [View](../operations/getIsWatchingIssueBulk.md) |
| GET | `/rest/api/3/issue/{issueIdOrKey}/watchers` | Get issue watchers | [View](../operations/getIssueWatchers.md) |
| POST | `/rest/api/3/issue/{issueIdOrKey}/watchers` | Add watcher | [View](../operations/addWatcher.md) |
| DELETE | `/rest/api/3/issue/{issueIdOrKey}/watchers` | Delete watcher | [View](../operations/removeWatcher.md) |
