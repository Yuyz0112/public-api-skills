# Issue links

This resource represents links between issues. Use it to get, create, and delete links between issues.

To use it, the site must have [issue linking](https://confluence.atlassian.com/x/yoXKM) enabled.

## Operations

| Method | Path | Summary | Details |
|--------|------|---------|----------|
| POST | `/rest/api/3/issueLink` | Create issue link | [View](../operations/linkIssues.md) |
| GET | `/rest/api/3/issueLink/{linkId}` | Get issue link | [View](../operations/getIssueLink.md) |
| DELETE | `/rest/api/3/issueLink/{linkId}` | Delete issue link | [View](../operations/deleteIssueLink.md) |
