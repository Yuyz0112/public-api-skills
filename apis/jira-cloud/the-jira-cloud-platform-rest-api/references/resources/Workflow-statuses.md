# Workflow statuses

This resource represents issue workflow statuses. Use it to obtain a list of all statuses associated with workflows and the details of a status.

## Operations

| Method | Path | Summary | Details |
|--------|------|---------|----------|
| GET | `/rest/api/3/status` | Get all statuses | [View](../operations/getStatuses.md) |
| GET | `/rest/api/3/status/{idOrName}` | Get status | [View](../operations/getStatus.md) |
