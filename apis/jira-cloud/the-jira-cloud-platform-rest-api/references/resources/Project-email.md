# Project email

This resource represents the email address used to send a project's notifications. Use it to get and set the [project's sender email address](https://confluence.atlassian.com/x/dolKLg).

## Operations

| Method | Path | Summary | Details |
|--------|------|---------|----------|
| GET | `/rest/api/3/project/{projectId}/email` | Get project's sender email | [View](../operations/getProjectEmail.md) |
| PUT | `/rest/api/3/project/{projectId}/email` | Set project's sender email | [View](../operations/updateProjectEmail.md) |
