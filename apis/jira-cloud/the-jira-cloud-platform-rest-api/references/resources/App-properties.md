# App properties

This resource represents app properties. Use it to store arbitrary data for your
[Connect app](https://developer.atlassian.com/cloud/jira/platform/index/#connect-apps).

## Operations

| Method | Path | Summary | Details |
|--------|------|---------|----------|
| GET | `/rest/atlassian-connect/1/addons/{addonKey}/properties` | Get app properties | [View](../operations/AddonPropertiesResource-getAddonProperties-get.md) |
| GET | `/rest/atlassian-connect/1/addons/{addonKey}/properties/{propertyKey}` | Get app property | [View](../operations/AddonPropertiesResource-getAddonProperty-get.md) |
| PUT | `/rest/atlassian-connect/1/addons/{addonKey}/properties/{propertyKey}` | Set app property | [View](../operations/AddonPropertiesResource-putAddonProperty-put.md) |
| DELETE | `/rest/atlassian-connect/1/addons/{addonKey}/properties/{propertyKey}` | Delete app property | [View](../operations/AddonPropertiesResource-deleteAddonProperty-delete.md) |
| GET | `/rest/forge/1/app/properties` | Get app property keys (Forge) | [View](../operations/getForgeAppPropertyKeys.md) |
| GET | `/rest/forge/1/app/properties/{propertyKey}` | Get app property (Forge) | [View](../operations/getForgeAppProperty.md) |
| PUT | `/rest/forge/1/app/properties/{propertyKey}` | Set app property (Forge) | [View](../operations/putForgeAppProperty.md) |
| DELETE | `/rest/forge/1/app/properties/{propertyKey}` | Delete app property (Forge) | [View](../operations/deleteForgeAppProperty.md) |
