# Application roles

This resource represents application roles. Use it to get details of an application role or all application roles.

## Operations

| Method | Path | Summary | Details |
|--------|------|---------|----------|
| GET | `/rest/api/3/applicationrole` | Get all application roles | [View](../operations/getAllApplicationRoles.md) |
| GET | `/rest/api/3/applicationrole/{key}` | Get application role | [View](../operations/getApplicationRole.md) |
