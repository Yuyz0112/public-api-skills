# Screen tab fields

This resource represents the screen tab fields used to record issue details. Use it to get, add, move, and remove fields from screen tabs.

## Operations

| Method | Path | Summary | Details |
|--------|------|---------|----------|
| GET | `/rest/api/3/screens/{screenId}/tabs/{tabId}/fields` | Get all screen tab fields | [View](../operations/getAllScreenTabFields.md) |
| POST | `/rest/api/3/screens/{screenId}/tabs/{tabId}/fields` | Add screen tab field | [View](../operations/addScreenTabField.md) |
| DELETE | `/rest/api/3/screens/{screenId}/tabs/{tabId}/fields/{id}` | Remove screen tab field | [View](../operations/removeScreenTabField.md) |
| POST | `/rest/api/3/screens/{screenId}/tabs/{tabId}/fields/{id}/move` | Move screen tab field | [View](../operations/moveScreenTabField.md) |
