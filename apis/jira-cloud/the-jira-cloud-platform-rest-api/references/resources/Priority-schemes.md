# Priority schemes

This resource represents issue priority schemes. Use it to get priority schemes and related information, and to create, update and delete priority schemes.

## Operations

| Method | Path | Summary | Details |
|--------|------|---------|----------|
| GET | `/rest/api/3/priorityscheme` | Get priority schemes | [View](../operations/getPrioritySchemes.md) |
| POST | `/rest/api/3/priorityscheme` | Create priority scheme | [View](../operations/createPriorityScheme.md) |
| POST | `/rest/api/3/priorityscheme/mappings` | Suggested priorities for mappings | [View](../operations/suggestedPrioritiesForMappings.md) |
| GET | `/rest/api/3/priorityscheme/priorities/available` | Get available priorities by priority scheme | [View](../operations/getAvailablePrioritiesByPriorityScheme.md) |
| PUT | `/rest/api/3/priorityscheme/{schemeId}` | Update priority scheme | [View](../operations/updatePriorityScheme.md) |
| DELETE | `/rest/api/3/priorityscheme/{schemeId}` | Delete priority scheme | [View](../operations/deletePriorityScheme.md) |
| GET | `/rest/api/3/priorityscheme/{schemeId}/priorities` | Get priorities by priority scheme | [View](../operations/getPrioritiesByPriorityScheme.md) |
| GET | `/rest/api/3/priorityscheme/{schemeId}/projects` | Get projects by priority scheme | [View](../operations/getProjectsByPriorityScheme.md) |
