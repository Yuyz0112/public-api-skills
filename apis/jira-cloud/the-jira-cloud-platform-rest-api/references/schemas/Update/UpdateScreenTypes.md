# UpdateScreenTypes

The IDs of the screens for the screen types of the screen scheme.

**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `create` | string | No | The ID of the create screen. To remove the screen association, pass a null. |
| `default` | string | No | The ID of the default screen. When specified, must include a screen ID as a default screen is required. |
| `edit` | string | No | The ID of the edit screen. To remove the screen association, pass a null. |
| `view` | string | No | The ID of the view screen. To remove the screen association, pass a null. |
