# SuggestedIssue

An issue suggested for use in the issue picker auto-completion.

**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `id` | integer (int64) | No | The ID of the issue. |
| `img` | string | No | The URL of the issue type's avatar. |
| `key` | string | No | The key of the issue. |
| `keyHtml` | string | No | The key of the issue in HTML format. |
| `summary` | string | No | The phrase containing the query string in HTML format, with the string highlighted with HTML bold tags. |
| `summaryText` | string | No | The phrase containing the query string, as plain text. |
