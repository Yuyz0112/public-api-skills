# AutoCompleteSuggestion

A field auto-complete suggestion.

**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `displayName` | string | No | The display name of a suggested item. If `fieldValue` or `predicateValue` are provided, the matching text is highlighted with the HTML bold tag. |
| `value` | string | No | The value of a suggested item. |
