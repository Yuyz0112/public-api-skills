# IssueTypeIdsToRemove

The list of issue type IDs to be removed from the field configuration scheme.

**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `issueTypeIds` | string[] | Yes | The list of issue type IDs. Must contain unique values not longer than 255 characters and not be empty. Maximum of 100 IDs. |
