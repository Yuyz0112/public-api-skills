# IssueTypeSchemeUpdateDetails

Details of the name, description, and default issue type for an issue type scheme.

**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `defaultIssueTypeId` | string | No | The ID of the default issue type of the issue type scheme. |
| `description` | string | No | The description of the issue type scheme. The maximum length is 4000 characters. |
| `name` | string | No | The name of the issue type scheme. The name must be unique. The maximum length is 255 characters. |
