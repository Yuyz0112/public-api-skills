# IssueTypeScheme

Details of an issue type scheme.

**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `defaultIssueTypeId` | string | No | The ID of the default issue type of the issue type scheme. |
| `description` | string | No | The description of the issue type scheme. |
| `id` | string | Yes | The ID of the issue type scheme. |
| `isDefault` | boolean | No | Whether the issue type scheme is the default. |
| `name` | string | Yes | The name of the issue type scheme. |
