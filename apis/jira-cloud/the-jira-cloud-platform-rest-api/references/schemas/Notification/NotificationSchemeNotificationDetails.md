# NotificationSchemeNotificationDetails

Details of a notification within a notification scheme.

**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `notificationType` | string | Yes | The notification type, e.g `CurrentAssignee`, `Group`, `EmailAddress`. |
| `parameter` | string | No | The value corresponding to the specified notification type. |
