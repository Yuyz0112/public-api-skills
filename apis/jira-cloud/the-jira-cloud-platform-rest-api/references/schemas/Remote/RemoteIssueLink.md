# RemoteIssueLink

Details of an issue remote link.

**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `application` | any | No | Details of the remote application the linked item is in. |
| `globalId` | string | No | The global ID of the link, such as the ID of the item on the remote system. |
| `id` | integer (int64) | No | The ID of the link. |
| `object` | any | No | Details of the item linked to. |
| `relationship` | string | No | Description of the relationship between the issue and the linked item. |
| `self` | string (uri) | No | The URL of the link. |
