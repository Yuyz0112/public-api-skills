# RemoteObject

The linked item.

**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `icon` | any | No | Details of the icon for the item. If no icon is defined, the default link icon is used in Jira. |
| `status` | any | No | The status of the item. |
| `summary` | string | No | The summary details of the item. |
| `title` | string | Yes | The title of the item. |
| `url` | string | Yes | The URL of the item. |
