# RemoteIssueLinkIdentifies

Details of the identifiers for a created or updated remote issue link.

**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `id` | integer (int64) | No | The ID of the remote issue link, such as the ID of the item on the remote system. |
| `self` | string | No | The URL of the remote issue link. |
