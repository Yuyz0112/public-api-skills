# SanitizedJqlQuery

Details of the sanitized JQL query.

**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `accountId` | string | No | The account ID of the user for whom sanitization was performed. |
| `errors` | any | No | The list of errors. |
| `initialQuery` | string | No | The initial query. |
| `sanitizedQuery` | string | No | The sanitized query, if there were no errors. |
