# HistoryMetadataParticipant

Details of user or system associated with a issue history metadata item.

**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `avatarUrl` | string | No | The URL to an avatar for the user or system associated with a history record. |
| `displayName` | string | No | The display name of the user or system associated with a history record. |
| `displayNameKey` | string | No | The key of the display name of the user or system associated with a history record. |
| `id` | string | No | The ID of the user or system associated with a history record. |
| `type` | string | No | The type of the user or system associated with a history record. |
| `url` | string | No | The URL of the user or system associated with a history record. |
