# HistoryMetadata

Details of issue history metadata.

**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `activityDescription` | string | No | The activity described in the history record. |
| `activityDescriptionKey` | string | No | The key of the activity described in the history record. |
| `actor` | any | No | Details of the user whose action created the history record. |
| `cause` | any | No | Details of the cause that triggered the creation the history record. |
| `description` | string | No | The description of the history record. |
| `descriptionKey` | string | No | The description key of the history record. |
| `emailDescription` | string | No | The description of the email address associated the history record. |
| `emailDescriptionKey` | string | No | The description key of the email address associated the history record. |
| `extraData` | object | No | Additional arbitrary information about the history record. |
| `generator` | any | No | Details of the system that generated the history record. |
| `type` | string | No | The type of the history record. |
