# SecuritySchemeWithProjects

Details about an issue security scheme.

**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `defaultLevel` | integer (int64) | No | The default level ID of the issue security scheme. |
| `description` | string | No | The description of the issue security scheme. |
| `id` | integer (int64) | Yes | The ID of the issue security scheme. |
| `name` | string | Yes | The name of the issue security scheme. |
| `projectIds` | integer[] | No | The list of project IDs associated with the issue security scheme. |
| `self` | string | Yes | The URL of the issue security scheme. |
