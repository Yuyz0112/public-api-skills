# VersionIssuesStatus

Counts of the number of issues in various statuses.

**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `done` | integer (int64) | No | Count of issues with status *done*. |
| `inProgress` | integer (int64) | No | Count of issues with status *in progress*. |
| `toDo` | integer (int64) | No | Count of issues with status *to do*. |
| `unmapped` | integer (int64) | No | Count of issues with a status other than *to do*, *in progress*, and *done*. |
