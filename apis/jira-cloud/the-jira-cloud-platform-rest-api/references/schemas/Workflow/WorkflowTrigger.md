# WorkflowTrigger

The trigger configuration associated with a workflow.

**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `id` | string | No | The ID of the trigger. |
| `parameters` | object | Yes | The parameters of the trigger. |
| `ruleKey` | string | Yes | The rule key of the trigger. |
