# WorkflowSchemeProjectAssociation

An associated workflow scheme and project.

**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `projectId` | string | Yes | The ID of the project. |
| `workflowSchemeId` | string | No | The ID of the workflow scheme. If the workflow scheme ID is `null`, the operation assigns the default workflow scheme. |
