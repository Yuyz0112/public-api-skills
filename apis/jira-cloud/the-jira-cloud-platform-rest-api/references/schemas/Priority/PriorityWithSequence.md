# PriorityWithSequence

An issue priority with sequence information.

**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `description` | string | No | The description of the issue priority. |
| `iconUrl` | string | No | The URL of the icon for the issue priority. |
| `id` | string | No | The ID of the issue priority. |
| `isDefault` | boolean | No | Whether this priority is the default. |
| `name` | string | No | The name of the issue priority. |
| `self` | string | No | The URL of the issue priority. |
| `sequence` | string | No | The sequence of the issue priority. |
| `statusColor` | string | No | The color used to indicate the issue priority. |
