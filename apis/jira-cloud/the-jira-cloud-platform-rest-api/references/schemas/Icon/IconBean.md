# IconBean

An icon.

**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `link` | string | No | The URL of the tooltip, used only for a status icon. |
| `title` | string | No | The title of the icon, for use as a tooltip on the icon. |
| `url16x16` | string | No | The URL of a 16x16 pixel icon. |
