# CustomFieldReplacement

Details about the replacement for a deleted version.

**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `customFieldId` | integer (int64) | No | The ID of the custom field in which to replace the version number. |
| `moveTo` | integer (int64) | No | The version number to use as a replacement for the deleted version. |
