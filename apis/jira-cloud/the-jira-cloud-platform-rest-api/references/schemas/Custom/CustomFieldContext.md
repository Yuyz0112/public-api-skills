# CustomFieldContext

The details of a custom field context.

**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `description` | string | Yes | The description of the context. |
| `id` | string | Yes | The ID of the context. |
| `isAnyIssueType` | boolean | Yes | Whether the context apply to all issue types. |
| `isGlobalContext` | boolean | Yes | Whether the context is global. |
| `name` | string | Yes | The name of the context. |
