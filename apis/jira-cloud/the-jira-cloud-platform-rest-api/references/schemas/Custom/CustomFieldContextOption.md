# CustomFieldContextOption

Details of the custom field options for a context.

**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `disabled` | boolean | Yes | Whether the option is disabled. |
| `id` | string | Yes | The ID of the custom field option. |
| `optionId` | string | No | For cascading options, the ID of the custom field option containing the cascading option. |
| `value` | string | Yes | The value of the custom field option. |
