## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `group` | any | No | The group subscribing to filter. |
| `id` | integer (int64) | No | The ID of the filter subscription. |
| `user` | any | No | The user subscribing to filter. |
