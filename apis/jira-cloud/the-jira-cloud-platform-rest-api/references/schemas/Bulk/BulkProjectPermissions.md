# BulkProjectPermissions

Details of project permissions and associated issues and projects to look up.

**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `issues` | integer[] | No | List of issue IDs. |
| `permissions` | string[] | Yes | List of project permissions. |
| `projects` | integer[] | No | List of project IDs. |
