# BulkContextualConfiguration

Details of the contextual configuration for a custom field.

**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `configuration` | any | No | The field configuration. |
| `customFieldId` | string | Yes | The ID of the custom field. |
| `fieldContextId` | string | Yes | The ID of the field context the configuration is associated with. |
| `id` | string | Yes | The ID of the configuration. |
| `schema` | any | No | The field value schema. |
