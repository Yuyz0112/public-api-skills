# RulePayload

The payload for creating rules in a workflow

**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `parameters` | object | No | The parameters of the rule |
| `ruleKey` | string | No | The key of the rule. See https://developer.atlassian.com/cloud/jira/platform/rest/v3/api-group-workflows/\#api-rest-api-3-workflows-capabilities-get |
