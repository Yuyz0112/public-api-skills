# AttachmentArchiveItemReadable

Metadata for an item in an attachment archive.

**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `index` | integer (int64) | No | The position of the item within the archive. |
| `label` | string | No | The label for the archive item. |
| `mediaType` | string | No | The MIME type of the archive item. |
| `path` | string | No | The path of the archive item. |
| `size` | string | No | The size of the archive item. |
