# Attachment

Details about an attachment.

**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `author` | any | No | Details of the user who added the attachment. |
| `content` | string | No | The content of the attachment. |
| `created` | string (date-time) | No | The datetime the attachment was created. |
| `filename` | string | No | The file name of the attachment. |
| `id` | string | No | The ID of the attachment. |
| `mimeType` | string | No | The MIME type of the attachment. |
| `self` | string | No | The URL of the attachment details response. |
| `size` | integer (int64) | No | The size of the attachment. |
| `thumbnail` | string | No | The URL of a thumbnail representing the attachment. |
