# AttachmentMetadata

Metadata for an issue attachment.

**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `author` | any | No | Details of the user who attached the file. |
| `content` | string | No | The URL of the attachment. |
| `created` | string (date-time) | No | The datetime the attachment was created. |
| `filename` | string | No | The name of the attachment file. |
| `id` | integer (int64) | No | The ID of the attachment. |
| `mimeType` | string | No | The MIME type of the attachment. |
| `properties` | object | No | Additional properties of the attachment. |
| `self` | string (uri) | No | The URL of the attachment metadata details. |
| `size` | integer (int64) | No | The size of the attachment. |
| `thumbnail` | string | No | The URL of a thumbnail representing the attachment. |
