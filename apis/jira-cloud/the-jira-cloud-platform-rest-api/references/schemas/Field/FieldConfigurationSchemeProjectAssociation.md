# FieldConfigurationSchemeProjectAssociation

Associated field configuration scheme and project.

**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `fieldConfigurationSchemeId` | string | No | The ID of the field configuration scheme. If the field configuration scheme ID is `null`, the operation assigns the default field configuration scheme. |
| `projectId` | string | Yes | The ID of the project. |
