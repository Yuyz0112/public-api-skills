# FieldAssociationsRequest

Details of field associations with projects.

**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `associationContexts` | AssociationContextObject[] | Yes | Contexts to associate/unassociate the fields with. |
| `fields` | FieldIdentifierObject[] | Yes | Fields to associate/unassociate with projects. |
