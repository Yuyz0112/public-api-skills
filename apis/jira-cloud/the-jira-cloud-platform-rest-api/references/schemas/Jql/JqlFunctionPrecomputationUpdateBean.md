# JqlFunctionPrecomputationUpdateBean

Precomputation id and its new value.

**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `error` | string | No | The error message to be displayed to the user if the given function clause is no longer valid during recalculation of the precomputation. |
| `id` | string | Yes | The id of the precomputation to update. |
| `value` | string | No | The new value of the precomputation. |
