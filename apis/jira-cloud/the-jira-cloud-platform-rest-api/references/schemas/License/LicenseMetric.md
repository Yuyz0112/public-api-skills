# LicenseMetric

A metric that provides insight into the active licence details

**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `key` | string | No | The key of a specific license metric. |
| `value` | string | No | The calculated value of a licence metric linked to the key. An example licence metric is the approximate number of user accounts. |
