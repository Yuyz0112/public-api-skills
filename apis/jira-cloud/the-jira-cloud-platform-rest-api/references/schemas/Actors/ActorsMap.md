# ActorsMap

**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `group` | string[] | No | The name of the group to add. This parameter cannot be used with the `groupId` parameter. As a group's name can change, use of `groupId` is recommended. |
| `groupId` | string[] | No | The ID of the group to add. This parameter cannot be used with the `group` parameter. |
| `user` | string[] | No | The user account ID of the user to add. |
