# ScreenTypes

The IDs of the screens for the screen types of the screen scheme.

**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `create` | integer (int64) | No | The ID of the create screen. |
| `default` | integer (int64) | Yes | The ID of the default screen. Required when creating a screen scheme. |
| `edit` | integer (int64) | No | The ID of the edit screen. |
| `view` | integer (int64) | No | The ID of the view screen. |
