# TimeTrackingDetails

Time tracking details.

**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `originalEstimate` | string | No | The original estimate of time needed for this issue in readable format. |
| `originalEstimateSeconds` | integer (int64) | No | The original estimate of time needed for this issue in seconds. |
| `remainingEstimate` | string | No | The remaining estimate of time needed for this issue in readable format. |
| `remainingEstimateSeconds` | integer (int64) | No | The remaining estimate of time needed for this issue in seconds. |
| `timeSpent` | string | No | Time worked on this issue in readable format. |
| `timeSpentSeconds` | integer (int64) | No | Time worked on this issue in seconds. |
