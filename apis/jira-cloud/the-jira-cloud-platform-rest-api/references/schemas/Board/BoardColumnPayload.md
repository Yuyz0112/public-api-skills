| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `maximumIssueConstraint` | integer (int64) | No | The maximum issue constraint for the column |
| `minimumIssueConstraint` | integer (int64) | No | The minimum issue constraint for the column |
| `name` | string | No | The name of the column |
| `statusIds` | ProjectCreateResourceIdentifier[] | No | The status IDs for the column |
