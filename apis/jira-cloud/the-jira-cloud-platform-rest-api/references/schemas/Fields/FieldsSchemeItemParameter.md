# FieldsSchemeItemParameter

The default parameters to apply to the field across all work types in the specified schemes, may be null if only work type-specific updates are needed

**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `description` | string | No | The custom description for the field, null to preserve current description |
| `isRequired` | boolean | No | Whether the field is required, null to preserve current requirement setting |
