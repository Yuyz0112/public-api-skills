# ProjectRoleGroup

Details of the group associated with the role.

**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `displayName` | string | No | The display name of the group. |
| `groupId` | string | No | The ID of the group. |
| `name` | string | No | The name of the group. As a group's name can change, use of `groupId` is recommended to identify the group. |
