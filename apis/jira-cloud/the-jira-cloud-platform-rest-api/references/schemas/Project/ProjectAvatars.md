# ProjectAvatars

List of project avatars.

**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `custom` | Avatar[] | No | List of avatars added to Jira. These avatars may be deleted. |
| `system` | Avatar[] | No | List of avatars included with Jira. These avatars cannot be deleted. |
