# DateRangeFilterRequest

List issues archived within a specified date range.

**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `dateAfter` | string | Yes | List issues archived after a specified date, passed in the YYYY-MM-DD format. |
| `dateBefore` | string | Yes | List issues archived before a specified date provided in the YYYY-MM-DD format. |
