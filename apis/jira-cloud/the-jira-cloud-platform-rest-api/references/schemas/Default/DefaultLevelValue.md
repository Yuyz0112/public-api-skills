# DefaultLevelValue

Details of scheme and new default level.

**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `defaultLevelId` | string | Yes | The ID of the issue security level to set as default for the specified scheme. Providing null will reset the default level. |
| `issueSecuritySchemeId` | string | Yes | The ID of the issue security scheme to set default level for. |
