# Locale

Details of a locale.

**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `locale` | string | No | The locale code. The Java the locale format is used: a two character language code (ISO 639), an underscore, and two letter country code (ISO 3166). For example, en\_US represents a locale of English (United States). Required on create. |
