# CreatedIssue

Details about a created issue or subtask.

**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `id` | string | No | The ID of the created issue or subtask. |
| `key` | string | No | The key of the created issue or subtask. |
| `self` | string | No | The URL of the created issue or subtask. |
| `transition` | any | No | The response code and messages related to any requested transition. |
| `watchers` | any | No | The response code and messages related to any requested watchers. |
