Lists of JQL reference data.

**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `jqlReservedWords` | string[] | No | List of JQL query reserved words. |
| `visibleFieldNames` | FieldReferenceData[] | No | List of fields usable in JQL queries. |
| `visibleFunctionNames` | FunctionReferenceData[] | No | List of functions usable in JQL queries. |
