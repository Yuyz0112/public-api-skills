# Status

The status of the item.

**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `icon` | any | No | Details of the icon representing the status. If not provided, no status icon displays in Jira. |
| `resolved` | boolean | No | Whether the item is resolved. If set to "true", the link to the issue is displayed in a strikethrough font, otherwise the link displays in normal font. |
