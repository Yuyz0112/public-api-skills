# api-insights-summary-stats

API Insights usage summary stats for an organization

**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `total_request_count` | integer (int64) | No | The total number of requests within the queried time period |
| `rate_limited_request_count` | integer (int64) | No | The total number of requests that were rate limited within the queried time period |
