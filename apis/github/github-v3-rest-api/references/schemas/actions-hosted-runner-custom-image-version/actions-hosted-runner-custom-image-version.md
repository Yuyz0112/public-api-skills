# actions-hosted-runner-custom-image-version

Provides details of a hosted runner custom image version

**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `version` | string | Yes | The version of image. |
| `state` | string | Yes | The state of image version. |
| `size_gb` | integer | Yes | Image version size in GB. |
| `created_on` | string | Yes | The creation date time of the image version. |
| `state_details` | string | Yes | The image version status details. |
