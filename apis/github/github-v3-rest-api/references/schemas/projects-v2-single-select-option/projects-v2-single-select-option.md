# projects-v2-single-select-option

An option for a single select field

**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `id` | string | Yes | The unique identifier of the option. |
| `name` | string | Yes | The display name of the option. |
| `color` | string | No | The color associated with the option. |
| `description` | string | No | A short description of the option. |
