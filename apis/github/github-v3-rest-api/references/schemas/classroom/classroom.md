# classroom

A GitHub Classroom classroom

**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `id` | integer | Yes | Unique identifier of the classroom. |
| `name` | string | Yes | The name of the classroom. |
| `archived` | boolean | Yes | Whether classroom is archived. |
| `organization` | [simple-classroom-organization](simple-classroom-organization.md) | Yes |  |
| `url` | string | Yes | The URL of the classroom on GitHub Classroom. |
