# public-ip

Provides details of Public IP for a GitHub-hosted larger runners

**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `enabled` | boolean | No | Whether public IP is enabled. |
| `prefix` | string | No | The prefix for the public IP. |
| `length` | integer | No | The length of the IP prefix. |
