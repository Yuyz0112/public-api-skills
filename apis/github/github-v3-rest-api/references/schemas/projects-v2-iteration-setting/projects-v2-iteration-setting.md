# projects-v2-iteration-setting

An iteration setting for an iteration field

**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `id` | string | Yes | The unique identifier of the iteration setting. |
| `title` | string | Yes | The iteration title. |
| `title_html` | string | No | The iteration title, rendered as HTML. |
| `duration` | number | No | The duration of the iteration in days. |
| `start_date` | string | No | The start date of the iteration. |
| `completed` | boolean | No | Whether the iteration has been completed. |
