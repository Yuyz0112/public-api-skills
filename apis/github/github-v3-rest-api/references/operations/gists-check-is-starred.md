# GET /gists/{gist_id}/star

**Resource:** [gists](../resources/gists.md)
**Check if a gist is starred**
**Operation ID:** `gists/check-is-starred`

## Responses

| Status | Description |
|--------|-------------|
| 204 | Response if gist is starred |
| 304 | (reference) |
| 403 | (reference) |
| 404 | Not Found if gist is not starred |
