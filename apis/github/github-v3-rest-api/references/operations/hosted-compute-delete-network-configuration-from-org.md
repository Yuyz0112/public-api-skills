# DELETE /orgs/{org}/settings/network-configurations/{network_configuration_id}

**Resource:** [hosted-compute](../resources/hosted-compute.md)
**Delete a hosted compute network configuration from an organization**
**Operation ID:** `hosted-compute/delete-network-configuration-from-org`

Deletes a hosted compute network configuration from an organization.

OAuth app tokens and personal access tokens (classic) need the `write:network_configurations` scope to use this endpoint.

## Responses

| Status | Description |
|--------|-------------|
| 204 | Response |
