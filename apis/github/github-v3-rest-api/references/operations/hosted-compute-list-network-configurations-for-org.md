# GET /orgs/{org}/settings/network-configurations

**Resource:** [hosted-compute](../resources/hosted-compute.md)
**List hosted compute network configurations for an organization**
**Operation ID:** `hosted-compute/list-network-configurations-for-org`

Lists all hosted compute network configurations configured in an organization.

OAuth app tokens and personal access tokens (classic) need the `read:network_configurations` scope to use this endpoint.

| Status | Description |

| 200 | Response |
