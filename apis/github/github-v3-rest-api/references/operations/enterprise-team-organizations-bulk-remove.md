# POST /enterprises/{enterprise}/teams/{enterprise-team}/organizations/remove

**Resource:** [enterprise-team-organizations](../resources/enterprise-team-organizations.md)
**Remove organization assignments**
**Operation ID:** `enterprise-team-organizations/bulk-remove`

Unassign an enterprise team from multiple organizations.

## Request Body

**Required:** Yes

**Content Types:** `application/json`

## Responses

| Status | Description |
|--------|-------------|
| 204 | Successfully unassigned the enterprise team from organizations. |
