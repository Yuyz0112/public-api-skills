# campaigns

Endpoints to manage campaigns via the REST API.

## Operations

| Method | Path | Summary | Details |
|--------|------|---------|----------|
| GET | `/orgs/{org}/campaigns` | List campaigns for an organization | [View](../operations/campaigns-list-org-campaigns.md) |
| POST | `/orgs/{org}/campaigns` | Create a campaign for an organization | [View](../operations/campaigns-create-campaign.md) |
| GET | `/orgs/{org}/campaigns/{campaign_number}` | Get a campaign for an organization | [View](../operations/campaigns-get-campaign-summary.md) |
| DELETE | `/orgs/{org}/campaigns/{campaign_number}` | Delete a campaign for an organization | [View](../operations/campaigns-delete-campaign.md) |
| PATCH | `/orgs/{org}/campaigns/{campaign_number}` | Update a campaign | [View](../operations/campaigns-update-campaign.md) |
