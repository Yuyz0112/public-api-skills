# oidc

Endpoints to manage GitHub OIDC configuration using the REST API.

## Operations

| Method | Path | Summary | Details |
|--------|------|---------|----------|
| GET | `/orgs/{org}/actions/oidc/customization/sub` | Get the customization template for an OIDC subject claim for an organization | [View](../operations/oidc-get-oidc-custom-sub-template-for-org.md) |
| PUT | `/orgs/{org}/actions/oidc/customization/sub` | Set the customization template for an OIDC subject claim for an organization | [View](../operations/oidc-update-oidc-custom-sub-template-for-org.md) |
