# CustomTypeCompact

Custom Types extend the types of Asana Objects, currently only Custom Tasks are supported.

**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `gid` | string | No | Globally unique identifier of the resource, as a string. |
| `resource_type` | string | No | The base type of this resource. |
| `name` | string | No | The name of the custom type. |
