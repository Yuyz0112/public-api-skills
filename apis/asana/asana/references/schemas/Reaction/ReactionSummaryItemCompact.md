# ReactionSummaryItemCompact

A summary of an emoji reaction on an object.

**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `emoji_base` | string | No | The emoji base character used in the reaction. |
| `variant` | string | No | The full emoji string used in the reaction. |
| `count` | number | No | The number of reactions with the emoji variant on the object. |
| `reacted` | boolean | No | Whether the current user has reacted with the emoji variant on the object. |
