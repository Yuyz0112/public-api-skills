# Preview

A collection of rich text that will be displayed as a preview to another app.

This is read-only except for a small group of whitelisted apps.

**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `fallback` | string | No | Some fallback text to display if unable to display the full preview. |
| `footer` | string | No | Text to display in the footer. |
| `header` | string | No | Text to display in the header. |
| `header_link` | string | No | Where the header will link to. |
| `html_text` | string | No | HTML formatted text for the body of the preview. |
| `text` | string | No | Text for the body of the preview. |
| `title` | string | No | Text to display as the title. |
| `title_link` | string | No | Where to title will link to. |
