# PlaceholderCompact

A *placeholder* object represents a placeholder in Asana that can be used to represent a placeholder user.

**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `gid` | string | No | Globally unique identifier of the resource, as a string. |
| `resource_type` | string | No | The base type of this resource. |
| `name` | string | No | The placeholder's name. |
