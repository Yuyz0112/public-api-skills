# MemberCompact

A *member* object represents either a team or user.

**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `gid` | string | No | Globally unique identifier of the resource, as a string. |
| `resource_type` | string | No | The type of the member (team or user) |
| `name` | string | No | The name of the member |
