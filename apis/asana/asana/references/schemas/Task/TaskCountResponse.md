# TaskCountResponse

A response object returned from the task count endpoint.

**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `num_tasks` | integer | No | The number of tasks in a project. |
| `num_incomplete_tasks` | integer | No | The number of incomplete tasks in a project. |
| `num_completed_tasks` | integer | No | The number of completed tasks in a project. |
| `num_milestones` | integer | No | The number of milestones in a project. |
| `num_incomplete_milestones` | integer | No | The number of incomplete milestones in a project. |
| `num_completed_milestones` | integer | No | The number of completed milestones in a project. |
