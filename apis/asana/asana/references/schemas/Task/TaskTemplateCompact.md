# TaskTemplateCompact

A *task template* is an object that allows new tasks to be created with a predefined setup.

**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `gid` | string | No | Globally unique identifier of the resource, as a string. |
| `resource_type` | string | No | The base type of this resource. |
| `name` | string | No | Name of the task template. |
