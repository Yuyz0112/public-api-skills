# Custom types

A custom type allows distinct categorizations of objects based on their specific purpose or functionality. Structuring objects through a defined custom type provides greater flexibility in work management allowing users to better match the object to their use case

## Operations

| Method | Path | Summary | Details |
|--------|------|---------|----------|
| GET | `/custom_types` | Get all custom types associated with an object | [View](../operations/getCustomTypes.md) |
| GET | `/custom_types/{custom_type_gid}` | Get a custom type | [View](../operations/getCustomType.md) |
