# Time periods

A time period is an object that represents a domain-scoped date range that can be set on [goals](/reference/goals).

## Operations

| Method | Path | Summary | Details |
|--------|------|---------|----------|
| GET | `/time_periods/{time_period_gid}` | Get a time period | [View](../operations/getTimePeriod.md) |
| GET | `/time_periods` | Get time periods | [View](../operations/getTimePeriods.md) |
