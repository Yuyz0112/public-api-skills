# User task lists

A user task list represents the tasks assigned to a particular user. This list is the user's [My Tasks](https://asana.com/guide/help/fundamentals/my-tasks) list.

## Operations

| Method | Path | Summary | Details |
|--------|------|---------|----------|
| GET | `/user_task_lists/{user_task_list_gid}` | Get a user task list | [View](../operations/getUserTaskList.md) |
| GET | `/users/{user_gid}/user_task_list` | Get a user's task list | [View](../operations/getUserTaskListForUser.md) |
