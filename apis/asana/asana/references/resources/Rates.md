# Rates

A rate object represents the rate of a resource for a parent object.

## Operations

| Method | Path | Summary | Details |
|--------|------|---------|----------|
| GET | `/rates` | Get multiple rates | [View](../operations/getRates.md) |
| POST | `/rates` | Create a rate | [View](../operations/createRate.md) |
| GET | `/rates/{rate_gid}` | Get a rate | [View](../operations/getRate.md) |
| PUT | `/rates/{rate_gid}` | Update a rate | [View](../operations/updateRate.md) |
| DELETE | `/rates/{rate_gid}` | Delete a rate | [View](../operations/deleteRate.md) |
