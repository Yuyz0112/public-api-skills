# Team memberships

This object determines if a user is a member of a team.

## Operations

| Method | Path | Summary | Details |
|--------|------|---------|----------|
| GET | `/team_memberships/{team_membership_gid}` | Get a team membership | [View](../operations/getTeamMembership.md) |
| GET | `/team_memberships` | Get team memberships | [View](../operations/getTeamMemberships.md) |
| GET | `/teams/{team_gid}/team_memberships` | Get memberships from a team | [View](../operations/getTeamMembershipsForTeam.md) |
| GET | `/users/{user_gid}/team_memberships` | Get memberships from a user | [View](../operations/getTeamMembershipsForUser.md) |
