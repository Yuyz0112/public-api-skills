# Portfolio memberships

This object determines if a user is a member of a portfolio.

## Operations

| Method | Path | Summary | Details |
|--------|------|---------|----------|
| GET | `/portfolio_memberships` | Get multiple portfolio memberships | [View](../operations/getPortfolioMemberships.md) |
| GET | `/portfolio_memberships/{portfolio_membership_gid}` | Get a portfolio membership | [View](../operations/getPortfolioMembership.md) |
| GET | `/portfolios/{portfolio_gid}/portfolio_memberships` | Get memberships from a portfolio | [View](../operations/getPortfolioMembershipsForPortfolio.md) |
