# Workspace memberships

This object determines if a user is a member of a workspace.

## Operations

| Method | Path | Summary | Details |
|--------|------|---------|----------|
| GET | `/workspace_memberships/{workspace_membership_gid}` | Get a workspace membership | [View](../operations/getWorkspaceMembership.md) |
| GET | `/users/{user_gid}/workspace_memberships` | Get workspace memberships for a user | [View](../operations/getWorkspaceMembershipsForUser.md) |
| GET | `/workspaces/{workspace_gid}/workspace_memberships` | Get the workspace memberships for a workspace | [View](../operations/getWorkspaceMembershipsForWorkspace.md) |
