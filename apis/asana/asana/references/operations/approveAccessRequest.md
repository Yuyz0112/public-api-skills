# POST /access_requests/{access_request_gid}/approve

**Resource:** [Access requests](../resources/Access-requests.md)
**Approve an access request**
**Operation ID:** `approveAccessRequest`

Approves an access request for a target object.

## Responses

| Status | Description |
|--------|-------------|
| 200 | Successfully approved an access request. |
| 400 | (reference) |
| 401 | (reference) |
| 403 | (reference) |
| 404 | (reference) |
| 500 | (reference) |
