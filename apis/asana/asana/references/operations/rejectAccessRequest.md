# POST /access_requests/{access_request_gid}/reject

**Resource:** [Access requests](../resources/Access-requests.md)
**Reject an access request**
**Operation ID:** `rejectAccessRequest`

Rejects an access request for a target object.

## Responses

| Status | Description |
|--------|-------------|
| 200 | Successfully rejected an access request. |
| 400 | (reference) |
| 401 | (reference) |
| 403 | (reference) |
| 404 | (reference) |
| 500 | (reference) |
