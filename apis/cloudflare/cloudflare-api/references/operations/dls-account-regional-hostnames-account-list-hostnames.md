# GET /zones/{zone_id}/addressing/regional_hostnames

**Resource:** [DLS Regional Services](../resources/DLS-Regional-Services.md)
**List Regional Hostnames**
**Operation ID:** `dls-account-regional-hostnames-account-list-hostnames`

List all Regional Hostnames within a zone.

## Responses

| Status | Description |
|--------|-------------|
| 200 | List hostnames response |
| 4XX | Failure to list hostnames |

## Security

- **api_email**
- **api_key**
