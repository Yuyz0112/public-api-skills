# DELETE /zones/{zone_id}/addressing/regional_hostnames/{hostname}

**Resource:** [DLS Regional Services](../resources/DLS-Regional-Services.md)
**Delete Regional Hostname**
**Operation ID:** `dls-account-regional-hostnames-account-delete-hostname`

Delete the region configuration for a specific Regional Hostname.

## Responses

| Status | Description |
|--------|-------------|
| 200 | Delete hostname response |
| 4XX | Failure to delete hostname |
