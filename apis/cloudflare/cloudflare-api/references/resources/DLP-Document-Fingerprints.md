# DLP Document Fingerprints

## Operations

| Method | Path | Summary | Details |
|--------|------|---------|----------|
| GET | `/accounts/{account_id}/dlp/document_fingerprints` | Retrieve data about all document fingerprints. | [View](../operations/dlp-document-fingerprints-read-all.md) |
| POST | `/accounts/{account_id}/dlp/document_fingerprints` | Creates a new document fingerprint. | [View](../operations/dlp-document-fingerprints-create.md) |
| GET | `/accounts/{account_id}/dlp/document_fingerprints/{document_fingerprint_id}` | Retrieve data about a specific document fingerprint. | [View](../operations/dlp-document-fingerprints-read.md) |
| PUT | `/accounts/{account_id}/dlp/document_fingerprints/{document_fingerprint_id}` | Uploads a new version for a document fingerprint. | [View](../operations/dlp-document-fingerprints-upload.md) |
| POST | `/accounts/{account_id}/dlp/document_fingerprints/{document_fingerprint_id}` | Update the attributes of a single document fingerprint. | [View](../operations/dlp-document-fingerprints-update.md) |
| DELETE | `/accounts/{account_id}/dlp/document_fingerprints/{document_fingerprint_id}` | Delete a single document fingerprint. | [View](../operations/dlp-document-fingerprints-delete.md) |
