# rulesets_SetCacheSettingsCustomCacheKeyUser

How to use characteristics of the request user agent in the cache key.

**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `device_type` | boolean | No | Whether to use the user agent's device type in the cache key. |
| `geo` | boolean | No | Whether to use the user agents's country in the cache key. |
| `lang` | boolean | No | Whether to use the user agent's language in the cache key. |
