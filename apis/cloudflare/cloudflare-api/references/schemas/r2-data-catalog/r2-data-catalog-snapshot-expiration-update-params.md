# r2-data-catalog_snapshot-expiration-update-params

Updates snapshot expiration configuration (all fields optional).

**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `max_snapshot_age` | string | No | Updates the maximum age for snapshots optionally. |
| `min_snapshots_to_keep` | integer (int64) | No | Updates the minimum number of snapshots to retain optionally. |
| `state` | any | No | Updates the state optionally. |
