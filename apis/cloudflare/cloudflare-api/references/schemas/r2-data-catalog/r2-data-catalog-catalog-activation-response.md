# r2-data-catalog_catalog-activation-response

Contains response from activating an R2 bucket as a catalog.

**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `id` | string (uuid) | Yes | Use this to uniquely identify the activated catalog. |
| `name` | string | Yes | Specifies the name of the activated catalog. |
