# teams-devices_policy_summary

The device settings profile assigned to this registration.

**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `default` | boolean | Yes | Whether the device settings profile is the default profile for the account. |
| `deleted` | boolean | Yes | Whether the device settings profile was deleted. |
| `id` | string | Yes | The ID of the device settings profile. |
| `name` | string | Yes | The name of the device settings profile. |
| `updated_at` | string | Yes | The RFC3339 timestamp of when the device settings profile last changed for the registration. |
