# realtimekit_Participant

Represents a participant.

**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `created_at` | string (date-time) | Yes | When this object was created. The time is returned in ISO format. |
| `custom_participant_id` | string | Yes | A unique participant ID generated by the client. |
| `id` | string (uuid) | Yes | ID of the participant. |
| `name` | string | No | Name of the participant. |
| `picture` | string (uri) | No | URL to a picture of the participant. |
| `preset_name` | string | Yes | Preset applied to the participant. |
| `updated_at` | string (date-time) | Yes | When this object was updated. The time is returned in ISO format. |
