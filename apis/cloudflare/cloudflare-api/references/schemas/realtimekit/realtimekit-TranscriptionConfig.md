# realtimekit_TranscriptionConfig

Transcription Configurations

**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `keywords` | string[] | No | Adds specific terms to improve accurate detection during transcription. |
| `language` | enum: en-US, en-IN, de... | No | Specifies the language code for transcription to ensure accurate results. |
| `profanity_filter` | boolean | No | Control the inclusion of offensive language in transcriptions. |
