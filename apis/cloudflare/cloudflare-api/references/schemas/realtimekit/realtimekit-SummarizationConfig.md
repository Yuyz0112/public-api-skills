# realtimekit_SummarizationConfig

Summary Config

**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `summary_type` | enum: general, team_meeting, sales_call... | No | Defines the style of the summary, such as general, team meeting, or sales call. |
| `text_format` | enum: plain_text, markdown | No | Determines the text format of the summary, such as plain text or markdown. |
| `word_limit` | integer | No | Sets the maximum number of words in the meeting summary. |
