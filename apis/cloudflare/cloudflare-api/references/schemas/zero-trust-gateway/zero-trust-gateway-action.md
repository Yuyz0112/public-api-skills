# zero-trust-gateway_action

Specify the action to perform when the associated traffic, identity, and device posture expressions either absent or evaluate to `true`.

**Type:** enum

## Values

- `on`
- `off`
- `allow`
- `block`
- `scan`
- `noscan`
- `safesearch`
- `ytrestricted`
- `isolate`
- `noisolate`
- `override`
- `l4_override`
- `egress`
- `resolve`
- `quarantine`
- `redirect`
