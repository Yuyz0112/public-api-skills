# access_user_seat_expiration_inactive_time

The amount of time a user seat is inactive before it expires. When the user seat exceeds the set time of inactivity, the user is removed as an active seat and no longer counts against your Teams seat count.  Minimum value for this setting is 1 month (730h). Must be in the format `300ms` or `2h45m`. Valid time units are: `ns`, `us` (or `Âµs`), `ms`, `s`, `m`, `h`.

**Type:** primitive
