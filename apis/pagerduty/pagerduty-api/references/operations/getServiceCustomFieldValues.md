# GET /services/{id}/custom_fields/values

**Resource:** [Services](../resources/Services.md)
**Get Custom Field Values**
**Operation ID:** `getServiceCustomFieldValues`

Get custom field values for a service.

Scoped OAuth requires: `services.read`


## Responses

| Status | Description |
|--------|-------------|
| 200 | The list of custom field values. |
| 403 | (reference) |
| 404 | (reference) |
| 500 | (reference) |
