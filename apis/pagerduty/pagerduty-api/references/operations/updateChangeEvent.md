# PUT /change_events/{id}

**Resource:** [Change Events](../resources/Change-Events.md)
**Update a Change Event**
**Operation ID:** `updateChangeEvent`

Update an existing Change Event

Scoped OAuth requires: `change_events.write`


## Request Body

The Change Event to be updated.

**Content Types:** `application/json`

## Responses

| Status | Description |
|--------|-------------|
| 200 | The Change Event that was updated. |
| 400 | (reference) |
| 401 | (reference) |
| 403 | (reference) |
| 404 | (reference) |
