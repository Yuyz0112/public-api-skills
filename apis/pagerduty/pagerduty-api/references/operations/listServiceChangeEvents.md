# GET /services/{id}/change_events

**Resource:** [Change Events](../resources/Change-Events.md)
**List Change Events for a service**
**Operation ID:** `listServiceChangeEvents`

List all of the existing Change Events for a service.

Scoped OAuth requires: `services.read`


## Responses

| Status | Description |
|--------|-------------|
| 200 | The array of Change Events returned by the query. |
| 400 | (reference) |
| 401 | (reference) |
| 403 | (reference) |
| 404 | (reference) |
