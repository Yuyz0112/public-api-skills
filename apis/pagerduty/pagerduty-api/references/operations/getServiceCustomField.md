# GET /services/custom_fields/{field_id}

**Resource:** [Service Custom Fields](../resources/Service-Custom-Fields.md)
**Get a Field**
**Operation ID:** `getServiceCustomField`

Show detailed information about a Custom Field for Services.

Scoped OAuth requires: `custom_fields.read`


## Responses

| Status | Description |
|--------|-------------|
| 200 | The field requested. |
| 403 | (reference) |
| 404 | (reference) |
| 500 | (reference) |
