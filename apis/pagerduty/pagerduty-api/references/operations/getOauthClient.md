# GET /webhook_subscriptions/oauth_clients/{id}

**Resource:** [Webhooks](../resources/Webhooks.md)
**Get an OAuth client**
**Operation ID:** `getOauthClient`

Get details of a specific OAuth client by ID.

Requires admin or owner role permissions.

|--------|-------------|
| 200 | OAuth client details |
| 401 | (reference) |
| 403 | (reference) |
