# GET /services/custom_fields/{field_id}/field_options/{field_option_id}

**Resource:** [Service Custom Fields](../resources/Service-Custom-Fields.md)
**Get a Field Option**
**Operation ID:** `getServiceCustomFieldOption`

Get a field option for a given field.

Scoped OAuth requires: `custom_fields.read`


## Responses

| Status | Description |
|--------|-------------|
| 200 | The requested field option. |
| 400 | (reference) |
| 403 | (reference) |
| 404 | (reference) |
| 500 | (reference) |
