# GET /services/custom_fields

**Resource:** [Service Custom Fields](../resources/Service-Custom-Fields.md)
**List Fields**
**Operation ID:** `listServiceCustomFields`

List Custom Fields available for Services.

Scoped OAuth requires: `custom_fields.read`


## Responses

| Status | Description |
|--------|-------------|
| 200 | A list of fields. |
| 400 | (reference) |
| 500 | (reference) |
