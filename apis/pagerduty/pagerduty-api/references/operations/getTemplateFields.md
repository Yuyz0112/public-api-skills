# GET /templates/fields

**Resource:** [Templates](../resources/Templates.md)
**List template fields**
**Operation ID:** `getTemplateFields`

Get a list of fields that can be used on the account templates.

Scoped OAuth requires: `templates.read`


## Responses

| Status | Description |
|--------|-------------|
| 200 | An array of template fields. |
| 400 | (reference) |
| 402 | (reference) |
| 403 | (reference) |
| 500 | (reference) |
