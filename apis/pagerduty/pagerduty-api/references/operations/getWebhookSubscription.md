# GET /webhook_subscriptions/{id}

**Resource:** [Webhooks](../resources/Webhooks.md)
**Get a webhook subscription**
**Operation ID:** `getWebhookSubscription`

Gets details about an existing webhook subscription.

Scoped OAuth requires: `webhook_subscriptions.read`


## Responses

| Status | Description |
|--------|-------------|
| 200 | The webhook subscription that was requested. |
| 400 | (reference) |
| 401 | (reference) |
| 403 | (reference) |
| 404 | (reference) |
