# ConferenceBridge

**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `conference_number` | string | No | The phone number of the conference call for the conference bridge. Phone numbers should be formatted like +1 415-555-1212,,,,1234#, where a comma (,) represents a one-second wait and pound (#) completes access code input. |
| `conference_url` | string (url) | No | An URL for the conference bridge. This could be a link to a web conference or Slack channel. |
