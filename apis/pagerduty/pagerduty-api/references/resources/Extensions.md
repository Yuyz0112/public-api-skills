# Extensions

Extensions are representations of Extension Schema objects that are attached to Services.


## Operations

| Method | Path | Summary | Details |
|--------|------|---------|----------|
| GET | `/extensions` | List extensions | [View](../operations/listExtensions.md) |
| POST | `/extensions` | Create an extension | [View](../operations/createExtension.md) |
| GET | `/extensions/{id}` | Get an extension | [View](../operations/getExtension.md) |
| PUT | `/extensions/{id}` | Update an extension | [View](../operations/updateExtension.md) |
| DELETE | `/extensions/{id}` | Delete an extension | [View](../operations/deleteExtension.md) |
| POST | `/extensions/{id}/enable` | Enable an extension | [View](../operations/enableExtension.md) |
