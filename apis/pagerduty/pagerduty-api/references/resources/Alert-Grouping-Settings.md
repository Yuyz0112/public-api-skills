# Alert Grouping Settings

Alert Grouping Settings allow you to configure how alerts in services are grouped together into incidents.


## Operations

| Method | Path | Summary | Details |
|--------|------|---------|----------|
| GET | `/alert_grouping_settings` | List alert grouping settings | [View](../operations/listAlertGroupingSettings.md) |
| POST | `/alert_grouping_settings` | Create an Alert Grouping Setting | [View](../operations/postAlertGroupingSettings.md) |
| GET | `/alert_grouping_settings/{id}` | Get an Alert Grouping Setting | [View](../operations/getAlertGroupingSetting.md) |
| PUT | `/alert_grouping_settings/{id}` | Update an Alert Grouping Setting | [View](../operations/putAlertGroupingSetting.md) |
| DELETE | `/alert_grouping_settings/{id}` | Delete an Alert Grouping Setting | [View](../operations/deleteAlertGroupingSetting.md) |
