# Log Entries

A log of all the events that happen to an Incident, and these are exposed as Log Entries.


## Operations

| Method | Path | Summary | Details |
|--------|------|---------|----------|
| GET | `/log_entries` | List log entries | [View](../operations/listLogEntries.md) |
| GET | `/log_entries/{id}` | Get a log entry | [View](../operations/getLogEntry.md) |
| PUT | `/log_entries/{id}/channel` | Update log entry channel information. | [View](../operations/updateLogEntryChannel.md) |
