# Add-ons

Developers can write their own functionality to insert into PagerDuty's UI.


## Operations

| Method | Path | Summary | Details |
|--------|------|---------|----------|
| GET | `/addons` | List installed Add-ons | [View](../operations/listAddon.md) |
| POST | `/addons` | Install an Add-on | [View](../operations/createAddon.md) |
| GET | `/addons/{id}` | Get an Add-on | [View](../operations/getAddon.md) |
| PUT | `/addons/{id}` | Update an Add-on | [View](../operations/updateAddon.md) |
| DELETE | `/addons/{id}` | Delete an Add-on | [View](../operations/deleteAddon.md) |
