# Extension Schemas

A PagerDuty extension vendor represents a specific type of outbound extension such as Generic Webhook, Slack, ServiceNow.


## Operations

| Method | Path | Summary | Details |
|--------|------|---------|----------|
| GET | `/extension_schemas` | List extension schemas | [View](../operations/listExtensionSchemas.md) |
| GET | `/extension_schemas/{id}` | Get an extension vendor | [View](../operations/getExtensionSchema.md) |
