# Maintenance Windows

A Maintenance Window is used to temporarily disable one or more Services for a set period of time.


## Operations

| Method | Path | Summary | Details |
|--------|------|---------|----------|
| GET | `/maintenance_windows` | List maintenance windows | [View](../operations/listMaintenanceWindows.md) |
| POST | `/maintenance_windows` | Create a maintenance window | [View](../operations/createMaintenanceWindow.md) |
| GET | `/maintenance_windows/{id}` | Get a maintenance window | [View](../operations/getMaintenanceWindow.md) |
| PUT | `/maintenance_windows/{id}` | Update a maintenance window | [View](../operations/updateMaintenanceWindow.md) |
| DELETE | `/maintenance_windows/{id}` | Delete or end a maintenance window | [View](../operations/deleteMaintenanceWindow.md) |
