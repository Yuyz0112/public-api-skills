# Licenses

Licenses are allocated to Users to allow for per-User access to PagerDuty functionality within an Account.


## Operations

| Method | Path | Summary | Details |
|--------|------|---------|----------|
| GET | `/license_allocations` | List License Allocations | [View](../operations/listLicenseAllocations.md) |
| GET | `/licenses` | List Licenses | [View](../operations/listLicenses.md) |
