# GET /admin/api/2020-07/articles/tags.json

**Resource:** [online-store/article](../resources/online-store-article.md)
**Retrieves a list of all the tags**
**Operation ID:** `deprecated_202007_get_articles_tags`

https://shopify.dev/docs/admin-api/rest/reference/online-store/article#tags-2020-07

## Parameters

| Name | In | Type | Required | Description |
|------|------|------|----------|-------------|
| `limit` | query | any | No | The maximum number of tags to retrieve. |
| `popular` | query | any | No | A flag for ordering retrieved tags. If present in the request, then the results will be ordered by popularity, starting with the most popular tag. |
