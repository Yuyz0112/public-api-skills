# POST /admin/api/2021-01/checkouts/{token}/payments.json

**Resource:** [sales-channels/payment](../resources/sales-channels-payment.md)
**Creates a payment on a checkout using the session ID returned by the card vault**
**Operation ID:** `deprecated_202101_create_checkouts_param_token_payments`

https://shopify.dev/docs/admin-api/rest/reference/sales-channels/payment#create_payment-2021-01

## Parameters

| Name | In | Type | Required | Description |
|------|------|------|----------|-------------|
| `token` | path | string | Yes | token |
| `amount
                  required` | query | any | No | The amount of the payment. |
| `request_details
                  required` | query | any | No | The details of the request, including the following attributes:
                    
                        ip_address: The IP address of the customer.
                        accept_language: The language preferences of the customer, in the same format as a standard Accept-Language request header.

                        user_agent: The user agent string for the customer's device. |
| `session_id
                  required` | query | any | No | A session ID provided by the card vault when creating a payment session. |
| `unique_token
                  required` | query | any | No | A unique idempotency token generated by your app. This can be any value, but must be unique across all payment requests. |

## Request Body

**Content Types:** `application/json`
