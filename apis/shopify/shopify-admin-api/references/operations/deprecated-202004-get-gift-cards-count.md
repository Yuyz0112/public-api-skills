# GET /admin/api/2020-04/gift_cards/count.json

**Resource:** [plus/giftcard](../resources/plus-giftcard.md)
**Retrieves a count of gift cards**
**Operation ID:** `deprecated_202004_get_gift_cards_count`

https://shopify.dev/docs/admin-api/rest/reference/plus/giftcard#count-2020-04

## Parameters

| Name | In | Type | Required | Description |
|------|------|------|----------|-------------|
| `status` | query | any | No | Count gift cards with a given status. Valid values:
                    
                        enabled: Count only enabled gift cards
                        disabled: Count only disabled gift cards |
