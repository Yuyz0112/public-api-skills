# online-store/theme

## Operations

| Method | Path | Summary | Details |
|--------|------|---------|----------|
| GET | `/admin/api/2020-01/themes.json` | Retrieves a list of themes. | [View](../operations/deprecated-202001-get-themes.md) |
| POST | `/admin/api/2020-01/themes.json` | Creates a theme by providing the public URL of a ZIP file that contains the theme.
          A new theme is always unpublished by default. To publish a theme when you create it, include
          "role": "main" in the POST request. The theme will be published only after all
          of its files have been extracted and stored by Shopify, which might take a couple of minutes. | [View](../operations/deprecated-202001-create-themes.md) |
| GET | `/admin/api/2020-01/themes/{theme_id}.json` | Retrieves a single theme. | [View](../operations/deprecated-202001-get-themes-param-theme-id.md) |
