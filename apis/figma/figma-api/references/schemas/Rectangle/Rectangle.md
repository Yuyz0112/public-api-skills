# Rectangle

A rectangle that expresses a bounding box in absolute coordinates.

**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `x` | number | Yes | X coordinate of top left corner of the rectangle. |
| `y` | number | Yes | Y coordinate of top left corner of the rectangle. |
| `width` | number | Yes | Width of the rectangle. |
| `height` | number | Yes | Height of the rectangle. |
