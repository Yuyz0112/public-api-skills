# ResponsePagination

If pagination is needed due to the length of the response, identifies the next and previous pages.

**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `prev_page` | string | No | A URL that calls the previous page of the response. |
| `next_page` | string | No | A URL that calls the next page of the response. |
