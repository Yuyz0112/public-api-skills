# BlendMode

This type is a string enum with the following possible values

Normal blends:
- `PASS_THROUGH` (only applicable to objects with children)
- `NORMAL`

Darken:
- `DARKEN`
- `MULTIPLY`
- `LINEAR_BURN`
- `COLOR_BURN`

Lighten:
- `LIGHTEN`
- `SCREEN`
- `LINEAR_DODGE`
- `COLOR_DODGE`

Contrast:
- `OVERLAY`
- `SOFT_LIGHT`
- `HARD_LIGHT`

Inversion:
- `DIFFERENCE`
- `EXCLUSION`

Component:
- `HUE`
- `SATURATION`
- `COLOR`
- `LUMINOSITY`

**Type:** enum
