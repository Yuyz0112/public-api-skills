# ExpressionFunction

Defines the list of operators available to use in an Expression.

**Type:** enum

## Values

- `ADDITION`
- `SUBTRACTION`
- `MULTIPLICATION`
- `DIVISION`
- `EQUALS`
- `NOT_EQUAL`
- `LESS_THAN`
- `LESS_THAN_OR_EQUAL`
- `GREATER_THAN`
- `GREATER_THAN_OR_EQUAL`
- `AND`
- `OR`
- `VAR_MODE_LOOKUP`
- `NEGATE`
- `NOT`
