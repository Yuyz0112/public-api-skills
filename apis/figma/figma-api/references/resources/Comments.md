# Comments

Interact with file comments.

## Operations

| Method | Path | Summary | Details |
|--------|------|---------|----------|
| GET | `/v1/files/{file_key}/comments` | Get comments in a file | [View](../operations/getComments.md) |
| POST | `/v1/files/{file_key}/comments` | Add a comment to a file | [View](../operations/postComment.md) |
| DELETE | `/v1/files/{file_key}/comments/{comment_id}` | Delete a comment | [View](../operations/deleteComment.md) |
