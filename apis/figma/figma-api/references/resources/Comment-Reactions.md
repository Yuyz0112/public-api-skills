# Comment Reactions

Interact with reactions to file comments.

## Operations

| Method | Path | Summary | Details |
|--------|------|---------|----------|
| GET | `/v1/files/{file_key}/comments/{comment_id}/reactions` | Get reactions for a comment | [View](../operations/getCommentReactions.md) |
| POST | `/v1/files/{file_key}/comments/{comment_id}/reactions` | Add a reaction to a comment | [View](../operations/postCommentReaction.md) |
| DELETE | `/v1/files/{file_key}/comments/{comment_id}/reactions` | Delete a reaction | [View](../operations/deleteCommentReaction.md) |
