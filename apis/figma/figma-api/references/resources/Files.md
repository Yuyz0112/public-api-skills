# Files

Get file JSON, images, and other file-related content.

## Operations

| Method | Path | Summary | Details |
|--------|------|---------|----------|
| GET | `/v1/files/{file_key}` | Get file JSON | [View](../operations/getFile.md) |
| GET | `/v1/files/{file_key}/nodes` | Get file JSON for specific nodes | [View](../operations/getFileNodes.md) |
| GET | `/v1/images/{file_key}` | Render images of file nodes | [View](../operations/getImages.md) |
| GET | `/v1/files/{file_key}/images` | Get image fills | [View](../operations/getImageFills.md) |
| GET | `/v1/files/{file_key}/meta` | Get file metadata | [View](../operations/getFileMeta.md) |
| GET | `/v1/files/{file_key}/versions` | Get versions of a file | [View](../operations/getFileVersions.md) |
