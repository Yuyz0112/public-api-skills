# Variables

Interact with variables in an Enterprise organization.

## Operations

| Method | Path | Summary | Details |
|--------|------|---------|----------|
| GET | `/v1/files/{file_key}/variables/local` | Get local variables | [View](../operations/getLocalVariables.md) |
| GET | `/v1/files/{file_key}/variables/published` | Get published variables | [View](../operations/getPublishedVariables.md) |
| POST | `/v1/files/{file_key}/variables` | Create/modify/delete variables | [View](../operations/postVariables.md) |
