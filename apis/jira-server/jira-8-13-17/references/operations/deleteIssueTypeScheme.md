# DELETE /issuetypescheme/{schemeId}

**Resource:** [issuetypescheme](../resources/issuetypescheme.md)
**Operation ID:** `deleteIssueTypeScheme`

Deletes the specified issue type scheme. Any projects associated with this IssueTypeScheme will be automatically
 associated with the global default IssueTypeScheme. (Admin required)

## Responses

| Status | Description |
|--------|-------------|
| 200 | Successful Response |
