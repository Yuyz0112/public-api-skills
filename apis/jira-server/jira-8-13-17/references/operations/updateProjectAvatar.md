# PUT /project/{projectIdOrKey}/avatar

**Resource:** [project](../resources/project.md)
**Operation ID:** `updateProjectAvatar`

Updates an avatar for a project. This is step 3/3 of changing an avatar for a project:

 <ul>
 <li>Upload (store temporary avatar)</li>
 <li>Crop (create avatar from temporary)</li>
 <li><b>Update (update project avatar)</b></li>
 </ul>

## Responses

| Status | Description |
|--------|-------------|
| 200 | Successful Response |
