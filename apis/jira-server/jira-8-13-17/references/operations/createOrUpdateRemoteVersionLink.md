# POST /version/{versionId}/remotelink/{globalId}

**Resource:** [version](../resources/version.md)
**Operation ID:** `createOrUpdateRemoteVersionLink`

Create a remote version link via POST.  The link's global ID will be taken from the
 JSON payload if provided; otherwise, it will be generated.

## Responses

| Status | Description |
|--------|-------------|
| 200 | Successful Response |
