# DELETE /workflowscheme/{id}/draft/workflow

**Resource:** [workflowscheme](../resources/workflowscheme.md)
**Operation ID:** `deleteDraftWorkflowMapping`

Delete the passed workflow from the draft workflow scheme.

## Parameters

| Name | In | Type | Required | Description |
|------|------|------|----------|-------------|
| `workflowName` | query | string | No | the name of the workflow to delete. |

## Responses

| Status | Description |
|--------|-------------|
| 200 | Successful Response |
