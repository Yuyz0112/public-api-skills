# POST /filter

**Resource:** [filter](../resources/filter.md)
**Operation ID:** `createFilter`

Creates a new filter, and returns newly created filter.
 Currently sets permissions just using the users default sharing permissions

## Parameters

| Name | In | Type | Required | Description |
|------|------|------|----------|-------------|
| `expand` | query | string | No | the parameters to expand |

## Responses

| Status | Description |
|--------|-------------|
| 200 | Successful Response |
