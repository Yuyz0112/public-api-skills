# GET /version/remotelink

**Resource:** [version](../resources/version.md)
**Operation ID:** `getRemoteVersionLinks`

Returns the remote version links for a given global ID.

## Parameters

| Name | In | Type | Required | Description |
|------|------|------|----------|-------------|
| `globalId` | query | string | No | the global ID of the remote resource that is linked to the versions |

## Responses

| Status | Description |
|--------|-------------|
| 200 | Successful Response |
