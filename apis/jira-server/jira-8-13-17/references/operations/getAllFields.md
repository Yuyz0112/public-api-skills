# GET /screens/{screenId}/tabs/{tabId}/fields

**Resource:** [screens](../resources/screens.md)
**Operation ID:** `getAllFields`

Gets all fields for a given tab

## Parameters

| Name | In | Type | Required | Description |
|------|------|------|----------|-------------|
| `projectKey` | query | string | No | the key of the project; this parameter is optional |

## Responses

| Status | Description |
|--------|-------------|
| 200 | Successful Response |
