# DELETE /issuetypescheme/{schemeId}/associations

**Resource:** [issuetypescheme](../resources/issuetypescheme.md)
**Operation ID:** `removeAllProjectAssociations`

Removes all project associations for the specified issue type scheme. These projects revert to an association with
 the default/global issue type scheme.(Admin required)

## Responses

| Status | Description |
|--------|-------------|
| 200 | Successful Response |
