# POST /project/{projectIdOrKey}/role/{id}

**Resource:** [project](../resources/project.md)
**Operation ID:** `addActorUsers`

Adds an actor (user or group) to a project role.
 <p>
 For user actors, their usernames should be used.

## Responses

| Status | Description |
|--------|-------------|
| 200 | Successful Response |
