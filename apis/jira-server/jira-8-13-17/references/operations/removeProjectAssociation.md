# DELETE /issuetypescheme/{schemeId}/associations/{projIdOrKey}

**Resource:** [issuetypescheme](../resources/issuetypescheme.md)
**Operation ID:** `removeProjectAssociation`

For the specified issue type scheme, removes the given project association . This project reverts to an association
 with the default/global issue type scheme.(Admin required)

## Responses

| Status | Description |
|--------|-------------|
| 200 | Successful Response |
