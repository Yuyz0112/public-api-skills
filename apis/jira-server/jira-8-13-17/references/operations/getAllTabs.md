# GET /screens/{screenId}/tabs

**Resource:** [screens](../resources/screens.md)
**Operation ID:** `getAllTabs`

Returns a list of all tabs for the given screen

## Parameters

| Name | In | Type | Required | Description |
|------|------|------|----------|-------------|
| `projectKey` | query | string | No | the key of the project; this parameter is optional |

## Responses

| Status | Description |
|--------|-------------|
| 200 | Successful Response |
