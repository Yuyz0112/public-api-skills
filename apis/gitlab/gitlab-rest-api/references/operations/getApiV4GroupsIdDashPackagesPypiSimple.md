**Resource:** [Packages](../resources/Packages.md)
**The PyPi Simple Group Index Endpoint**

## Parameters

| Name | In | Type | Required | Description |
|------|------|------|----------|-------------|
| `id` | path | any | Yes | The ID or full path of the group. |

|--------|-------------|
