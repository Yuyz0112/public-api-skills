# POST /api/v4/users/{id}/activate

**Resource:** [Users](../resources/Users.md)
**Activate a deactivated user. Available only for admins.**
**Operation ID:** `postApiV4UsersIdActivate`

## Parameters

| Name | In | Type | Required | Description |
|------|------|------|----------|-------------|
| `id` | path | integer | Yes | The ID of the user |

## Responses

| Status | Description |
|--------|-------------|
| 201 | Created |
