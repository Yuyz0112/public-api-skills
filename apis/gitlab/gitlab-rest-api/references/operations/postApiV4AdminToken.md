# POST /api/v4/admin/token

**Resource:** [Admin](../resources/Admin.md)
**Get information about a token.**
**Operation ID:** `postApiV4AdminToken`

This feature was introduced in GitLab 17.5.

## Request Body

**Required:** Yes

**Content Types:** `application/json`

## Responses

| Status | Description |
|--------|-------------|
| 401 | Unauthorized |
| 403 | Forbidden |
| 404 | Not Found |
| 422 | Unprocessable |
