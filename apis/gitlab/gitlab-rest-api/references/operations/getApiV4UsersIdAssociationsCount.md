# GET /api/v4/users/{id}/associations_count

**Resource:** [Users](../resources/Users.md)
**Returns a list of a specified user's count of projects, groups, issues and merge requests.**
**Operation ID:** `getApiV4UsersIdAssociationsCount`

## Parameters

| Name | In | Type | Required | Description |
|------|------|------|----------|-------------|
| `id` | path | integer | Yes | ID of the user to query. |

## Responses

| Status | Description |
|--------|-------------|
| 200 | OK |
