# DELETE /api/v4/projects/{id}/pages/domains/{domain}

**Resource:** [Gitlab pages](../resources/Gitlab-pages.md)
**Delete a pages domain**
**Operation ID:** `deleteApiV4ProjectsIdPagesDomainsDomain`

## Parameters

| Name | In | Type | Required | Description |
|------|------|------|----------|-------------|
| `id` | path | any | Yes | The ID or URL-encoded path of the project owned by the authenticated user |
| `domain` | path | string | Yes | The domain |

## Responses

| Status | Description |
|--------|-------------|
| 204 | No Content |
