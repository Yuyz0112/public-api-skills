# GET /api/v4/groups/{id}/avatar

**Resource:** [Avatars](../resources/Avatars.md)
**Download the group avatar**
**Operation ID:** `getApiV4GroupsIdAvatar`

This feature was introduced in GitLab 14.0

## Parameters

| Name | In | Type | Required | Description |
|------|------|------|----------|-------------|
| `id` | path | string | Yes | The ID of the group |

## Responses

| Status | Description |
|--------|-------------|
| 200 | OK |
