# GET /api/v4/projects/{id}/avatar

**Resource:** [Avatars](../resources/Avatars.md)
**Download a project avatar**
**Operation ID:** `getApiV4ProjectsIdAvatar`

This feature was introduced in GitLab 16.9

## Parameters

| Name | In | Type | Required | Description |
|------|------|------|----------|-------------|
| `id` | path | any | Yes | ID or URL-encoded path of the project |

## Responses

| Status | Description |
|--------|-------------|
| 200 | OK |
