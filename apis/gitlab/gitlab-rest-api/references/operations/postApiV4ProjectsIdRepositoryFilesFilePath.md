# POST /api/v4/projects/{id}/repository/files/{file_path}

**Resource:** [Files](../resources/Files.md)
**Create new file in repository**
**Operation ID:** `postApiV4ProjectsIdRepositoryFilesFilePath`

## Parameters

| Name | In | Type | Required | Description |
|------|------|------|----------|-------------|
| `id` | path | string | Yes | The project ID |
| `file` | path | API::Validations::Types::WorkhorseFile | Yes | The file content to be created (generated by Multipart middleware) |
| `file_path` | path | string | Yes | The URL-encoded path to the file. |

## Request Body

**Required:** Yes

**Content Types:** `multipart/form-data`

## Responses

| Status | Description |
|--------|-------------|
| 201 | Created |
