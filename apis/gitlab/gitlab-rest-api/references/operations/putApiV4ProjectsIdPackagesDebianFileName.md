# PUT /api/v4/projects/{id}/packages/debian/{file_name}

**Resource:** [Packages](../resources/Packages.md)
**Upload Debian package**
**Operation ID:** `putApiV4ProjectsIdPackagesDebianFileName`

This feature was introduced in GitLab 14.0

## Parameters

| Name | In | Type | Required | Description |
|------|------|------|----------|-------------|
| `id` | path | any | Yes | The ID or URL-encoded path of the project |
| `file_name` | path | string | Yes | The filename |
| `file` | path | API::Validations::Types::WorkhorseFile | Yes | The package file to be published (generated by Multipart middleware) |

## Request Body

**Required:** Yes

**Content Types:** `multipart/form-data`

## Responses

| Status | Description |
|--------|-------------|
| 201 | Created |
| 400 | Bad Request |
| 401 | Unauthorized |
| 403 | Forbidden |
| 404 | Not Found |
