# GET /api/v4/groups/{id}/placeholder_reassignments

**Resource:** [groups](../resources/groups.md)
**Download the list of pending placeholder assignments for a group**
**Operation ID:** `getApiV4GroupsIdPlaceholderReassignments`

This feature was added in GitLab 17.10

## Parameters

| Name | In | Type | Required | Description |
|------|------|------|----------|-------------|
| `id` | path | string | Yes | The ID of a group |

## Responses

| Status | Description |
|--------|-------------|
| 200 | OK |
