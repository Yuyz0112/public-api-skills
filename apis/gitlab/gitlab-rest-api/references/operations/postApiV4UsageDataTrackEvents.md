# POST /api/v4/usage_data/track_events

**Resource:** [Usage data](../resources/Usage-data.md)
**Track multiple gitlab internal events**

**Required:** Yes

**Content Types:** `application/json`

| Status | Description |
|--------|-------------|
| 200 | OK |
