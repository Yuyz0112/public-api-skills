# PATCH /api/v4/groups/{id}/enterprise_users/{user_id}/disable_two_factor

**Resource:** [Group enterprise users](../resources/Group-enterprise-users.md)
**Disable two factor authentication for an enterprise user**
**Operation ID:** `patchApiV4GroupsIdEnterpriseUsersUserIdDisableTwoFactor`

## Parameters

| Name | In | Type | Required | Description |
|------|------|------|----------|-------------|
| `id` | path | any | Yes | The ID or URL-encoded path of the group |
| `user_id` | path | integer | Yes | ID of user account. |

## Responses

| Status | Description |
|--------|-------------|
| 204 | No Content |
