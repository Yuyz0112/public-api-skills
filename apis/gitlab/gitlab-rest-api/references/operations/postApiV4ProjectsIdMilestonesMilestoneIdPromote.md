# POST /api/v4/projects/{id}/milestones/{milestone_id}/promote

**Resource:** [Milestones](../resources/Milestones.md)
**Promote a milestone to group milestone**
**Operation ID:** `postApiV4ProjectsIdMilestonesMilestoneIdPromote`

This feature was introduced in GitLab 11.9

## Parameters

| Name | In | Type | Required | Description |
|------|------|------|----------|-------------|
| `id` | path | any | Yes | The ID or URL-encoded path of the project |

## Responses

| Status | Description |
|--------|-------------|
| 201 | Created |
