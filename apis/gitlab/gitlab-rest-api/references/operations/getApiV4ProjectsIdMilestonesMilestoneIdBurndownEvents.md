# GET /api/v4/projects/{id}/milestones/{milestone_id}/burndown_events

**Resource:** [Milestones](../resources/Milestones.md)
**Get a list of burndown events**
**Operation ID:** `getApiV4ProjectsIdMilestonesMilestoneIdBurndownEvents`

This feature was introduced in GitLab 12.1.

## Parameters

| Name | In | Type | Required | Description |
|------|------|------|----------|-------------|
| `id` | path | any | Yes | The ID or URL-encoded path of the project |

## Responses

| Status | Description |
|--------|-------------|
| 200 | OK |
