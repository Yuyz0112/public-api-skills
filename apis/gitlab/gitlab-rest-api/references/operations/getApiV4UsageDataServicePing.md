# GET /api/v4/usage_data/service_ping

**Resource:** [Usage data](../resources/Usage-data.md)
**Get the latest ServicePing payload**
**Operation ID:** `getApiV4UsageDataServicePing`

Introduces in Gitlab 16.9. Requires personal access token with read_service_ping scope.

## Responses

| Status | Description |
|--------|-------------|
| 200 | OK |
| 401 | 401 Unauthorized |
| 403 | Forbidden |
| 404 | Not found |
