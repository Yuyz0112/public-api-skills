# DELETE /api/v4/projects/{id}/boards/{board_id}

**Resource:** [Boards](../resources/Boards.md)
**Delete a project board**
**Operation ID:** `deleteApiV4ProjectsIdBoardsBoardId`

This feature was introduced in 10.4

## Parameters

| Name | In | Type | Required | Description |
|------|------|------|----------|-------------|
| `id` | path | any | Yes | The ID or URL-encoded path of the project |

## Responses

| Status | Description |
|--------|-------------|
| 204 | No Content |
