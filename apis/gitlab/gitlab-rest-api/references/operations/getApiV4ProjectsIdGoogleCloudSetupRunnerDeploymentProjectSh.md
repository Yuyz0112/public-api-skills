# GET /api/v4/projects/{id}/google_cloud/setup/runner_deployment_project.sh

**Resource:** [Project Google Cloud integration](../resources/Project-Google-Cloud-integration.md)
**Get shell script to set up Google Cloud project for runner deployment**
**Operation ID:** `getApiV4ProjectsIdGoogleCloudSetupRunnerDeploymentProjectSh`

This feature is experimental.

## Parameters

| Name | In | Type | Required | Description |
|------|------|------|----------|-------------|
| `id` | path | any | Yes | The ID or URL-encoded path of the project |
| `google_cloud_project_id` | query | string | Yes | ID of the Google Cloud project |

## Responses

| Status | Description |
|--------|-------------|
| 200 | OK |
