# Support pins

Operations related to support pins.

## Operations

| Method | Path | Summary | Details |
|--------|------|---------|----------|
| GET | `/api/v4/users/{id}/support_pin` | Get support PIN for a user. Available only for admins. | [View](../operations/getApiV4UsersIdSupportPin.md) |
| POST | `/api/v4/users/{id}/support_pin/revoke` | Revoke support PIN for a user. Available only for admins. | [View](../operations/postApiV4UsersIdSupportPinRevoke.md) |
| GET | `/api/v4/user/support_pin` | Get the current Support PIN for the authenticated user | [View](../operations/getApiV4UserSupportPin.md) |
| POST | `/api/v4/user/support_pin` | Create a new Support PIN for the authenticated user | [View](../operations/postApiV4UserSupportPin.md) |
