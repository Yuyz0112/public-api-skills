## Fields

| Field | Type | Required | Description |

| `play_from_tap` | integer | No | Tracks the number of times a video or media is played from a user tap |
| `playback25` | integer | No | Tracks the number of times a video reaches 25% of its duration |
| `playback50` | integer | No | Tracks the number of times a video reaches 50% of its duration |
| `playback75` | integer | No | Tracks the number of times a video reaches 75% of its duration |
| `playback_complete` | integer | No | Tracks the number of times a video is played to completion |
| `playback_start` | integer | No | Tracks the number of times a video playback is initiated |
| `video_views` | integer | No | Tracks the number of times a video is viewed |
| `watch_time_ms` | integer | No | Tracks the total time spent watching a video, measured in milliseconds |
