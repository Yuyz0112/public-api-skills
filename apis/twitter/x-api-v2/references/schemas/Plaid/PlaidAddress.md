# PlaidAddress

Address information for the account holder.

**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `city` | string | Yes | The city of the address. |
| `country` | string | Yes | The country of the address (ISO 3166-1 alpha-2 code). |
| `line1` | string | Yes | The first line of the address. |
| `line2` | string | No | The second line of the address. |
| `postalCode` | string | No | The postal code of the address. |
| `region` | string | No | The region or state of the address. |
