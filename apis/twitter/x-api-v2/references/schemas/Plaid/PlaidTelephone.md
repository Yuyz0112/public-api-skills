# PlaidTelephone

Telephone information for the account holder.

**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `country` | string | Yes | The country code for the phone number (e.g., '+1'). |
| `number` | string | Yes | The phone number. |
| `type` | string | Yes | The type of phone number (e.g., 'mobile'). |
