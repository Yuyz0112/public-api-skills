Describes a choice in a Poll object.

**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `label` | [PollOptionLabel](PollOptionLabel.md) | Yes |  |
| `position` | integer | Yes | Position of this choice in the poll. |
| `votes` | integer | Yes | Number of users who voted for this choice. |
